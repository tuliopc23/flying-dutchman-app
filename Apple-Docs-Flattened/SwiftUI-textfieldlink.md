# TextFieldLink
A control that requests text input from the user when pressed.
watchOS 9.0+
```
struct TextFieldLink<Label> where Label : [View](https://developer.apple.com/documentation/swiftui/view)
```

## [Overview](https://developer.apple.com/documentation/swiftui/textfieldlink#overview)
A `TextFieldLink` should be used to request text input from the user through a button interface.
