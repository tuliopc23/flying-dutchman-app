# init(isOn:label:)
Creates a toggle that displays a custom label.
iOS 13.0+iPadOS 13.0+Mac Catalyst 13.0+macOS 10.15+tvOS 13.0+visionOS 1.0+watchOS 6.0+
```
init(
    isOn: [Binding](https://developer.apple.com/documentation/swiftui/binding)<[Bool](https://developer.apple.com/documentation/Swift/Bool)>,
    @[ViewBuilder](https://developer.apple.com/documentation/swiftui/viewbuilder) label: () -> Label
)
```

## [ Parameters ](https://developer.apple.com/documentation/swiftui/toggle/init\(ison:label:\)#parameters) 

`isOn`
    
A binding to a property that determines whether the toggle is on or off. 

`label`
    
A view that describes the purpose of the toggle.
