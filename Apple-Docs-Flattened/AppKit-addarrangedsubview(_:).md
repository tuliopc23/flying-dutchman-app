# addArrangedSubview(_:)
Adds a view as an arranged split pane.
macOS 10.11+
```
@[MainActor](https://developer.apple.com/documentation/Swift/MainActor)
func addArrangedSubview(_ view: [NSView](https://developer.apple.com/documentation/appkit/nsview))
```

## [Discussion](https://developer.apple.com/documentation/appkit/nssplitview/addarrangedsubview\(_:\)#Discussion)
If the view isn’t a subview of the split view, calling this method adds it to the split view’s [`subviews`](https://developer.apple.com/documentation/appkit/nsview/subviews) array.
