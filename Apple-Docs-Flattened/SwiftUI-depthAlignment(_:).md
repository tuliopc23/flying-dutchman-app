# depthAlignment(_:)
Sets the depth alignment for this layout.
visionOS 26.0+
```
func depthAlignment(_ alignment: [DepthAlignment](https://developer.apple.com/documentation/swiftui/depthalignment)) -> some [Layout](https://developer.apple.com/documentation/swiftui/layout)

```

## [ Parameters ](https://developer.apple.com/documentation/SwiftUI/Layout/depthAlignment\(_:\)#parameters) 

`alignment`
    
A [`DepthAlignment`](https://developer.apple.com/documentation/swiftui/depthalignment) value to use for aligning layout’s subviews
## [Discussion](https://developer.apple.com/documentation/SwiftUI/Layout/depthAlignment\(_:\)#discussion)
Use `depthAlignment(_:)` to specify a depth guide for aligning subviews of this layout.
In the example below, the button to play the robot animation is aligned to the `.front` of the `HStack`.
```
   let depthStack = HStackLayout().depthAlignment(.front)
   depthStack {
       RobotModel()
       Button("Play animation") {
           playRobotAnimation()
       }
       .glassBackgroundEffect()
   }

```
