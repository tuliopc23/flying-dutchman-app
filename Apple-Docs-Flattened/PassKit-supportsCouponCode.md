# supportsCouponCode
A Boolean value that determines whether the payment sheet displays the coupon code field.
iOS 15.0+iPadOS 15.0+Mac Catalyst 15.0+macOS 12.0+visionOS 1.0+
```
var supportsCouponCode: [Bool](https://developer.apple.com/documentation/Swift/Bool) { get set }
```

## [Discussion](https://developer.apple.com/documentation/PassKit/PKPaymentRequest/supportsCouponCode#Discussion)
Set the value to `true` to display the coupon code field.
