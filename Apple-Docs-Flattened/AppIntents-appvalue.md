# AppValue
An interface that describes conceptual types you use in app intents.
iOS 16.0+iPadOS 16.0+Mac CatalystmacOS 13.0+tvOS 16.0+visionOSwatchOS 9.0+
```
protocol AppValue : [PersistentlyIdentifiable](https://developer.apple.com/documentation/appintents/persistentlyidentifiable), [TypeDisplayRepresentable](https://developer.apple.com/documentation/appintents/typedisplayrepresentable), _IntentValue, [Sendable](https://developer.apple.com/documentation/Swift/Sendable)
```

## [Overview](https://developer.apple.com/documentation/appintents/appvalue#overview)
This protocol serves as the base type for conceptual interfaces like [`AppEntity`](https://developer.apple.com/documentation/appintents/appentity) or [`AppEnum`](https://developer.apple.com/documentation/appintents/appenum).
