visionOS 1.0+

```
struct ImageAnchor
```

## [Topics](https://developer.apple.com/documentation/arkit/imageanchor#topics)
### [Getting image information](https://developer.apple.com/documentation/arkit/imageanchor#Getting-image-information)
[`var originFromAnchorTransform: simd_float4x4`](https://developer.apple.com/documentation/arkit/imageanchor/originfromanchortransform)
The location and orientation of the image in world space.
[`var referenceImage: ReferenceImage`](https://developer.apple.com/documentation/arkit/imageanchor/referenceimage)
The reference image that this image anchor tracks.
[`var estimatedScaleFactor: Float`](https://developer.apple.com/documentation/arkit/imageanchor/estimatedscalefactor)
The estimated scale factor between the tracked image’s physical size and the reference image’s size.
[`var isTracked: Bool`](https://developer.apple.com/documentation/arkit/imageanchor/istracked)
A Boolean value that indicates whether ARKit is currently tracking this image.
[`var description: String`](https://developer.apple.com/documentation/arkit/imageanchor/description)
A textual representation of this anchor.
[`var id: UUID`](https://developer.apple.com/documentation/arkit/imageanchor/id)
The unique identifier of this anchor.
