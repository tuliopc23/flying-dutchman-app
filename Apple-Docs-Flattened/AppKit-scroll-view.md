# Scroll View
Provide an interface for navigating content that is too large to fit in the available space.
## [Topics](https://developer.apple.com/documentation/appkit/scroll-view#topics)
### [Views](https://developer.apple.com/documentation/appkit/scroll-view#Views)
[`class NSScrollView`](https://developer.apple.com/documentation/appkit/nsscrollview)
A view that displays a portion of a document view and provides scroll bars that allow the user to move the document view within the scroll view.
[`class NSScroller`](https://developer.apple.com/documentation/appkit/nsscroller)
An object that controls scrolling of a document view within a scroll view or other type of container view.
[`class NSClipView`](https://developer.apple.com/documentation/appkit/nsclipview)
An object that clips a document view to a scroll view’s frame.
### [Rulers](https://developer.apple.com/documentation/appkit/scroll-view#Rulers)
[`class NSRulerView`](https://developer.apple.com/documentation/appkit/nsrulerview)
A ruler and the markers above or to the side of a scroll view’s document view.
[`class NSRulerMarker`](https://developer.apple.com/documentation/appkit/nsrulermarker)
A symbol on a ruler view, indicating a location for the graphics element it represents in the client of the ruler view.
## [See Also](https://developer.apple.com/documentation/appkit/scroll-view#see-also)
### [Container views](https://developer.apple.com/documentation/appkit/scroll-view#Container-views)
[Localization-friendly layouts in macOS](https://developer.apple.com/documentation/Xcode/localization-friendly-layouts-in-macos)
This project demonstrates localization-friendly auto layout constraints.
[API Reference Grid View](https://developer.apple.com/documentation/appkit/grid-view)
Arrange views in a flexible grid, and handle the layout associated with those views.
[`class NSSplitView`](https://developer.apple.com/documentation/appkit/nssplitview)
A view that arranges two or more views in a linear stack running horizontally or vertically.
[Organize Your User Interface with a Stack View](https://developer.apple.com/documentation/appkit/organize-your-user-interface-with-a-stack-view)
Group individual views in your app’s user interface into a scrollable stack view.
[`class NSStackView`](https://developer.apple.com/documentation/appkit/nsstackview)
A view that arranges an array of views horizontally or vertically and updates their placement and sizing when the window size changes.
[`class NSTabView`](https://developer.apple.com/documentation/appkit/nstabview)
A multipage interface that displays one page at a time.
