# Enumerations
Enumerations for use with multiple classes.
## [Topics](https://developer.apple.com/documentation/appkit/enumerations#topics)
### [Enumerations](https://developer.apple.com/documentation/appkit/enumerations#Enumerations)
[`enum FrameResizePosition`](https://developer.apple.com/documentation/appkit/nscursor/frameresizeposition)
The position along the perimeter of a rectangular frame (its edges and corners) from which it’s resized.
[`enum NSHorizontalDirection`](https://developer.apple.com/documentation/appkit/nshorizontaldirection)
An absolute direction on the horizontal axis.
[`enum NSSharingCollaborationMode`](https://developer.apple.com/documentation/appkit/nssharingcollaborationmode)
Represents the types of sharing (collaborating on an item vs. sending a copy of the item) The share picker supports up to two modes, each of which corresponds to one of these types
[`enum DynamicRange`](https://developer.apple.com/documentation/appkit/nsimage/dynamicrange)
Describes how High Dynamic Range (HDR) image content displays.
[`enum NSTextCursorAccessoryPlacement`](https://developer.apple.com/documentation/appkit/nstextcursoraccessoryplacement)
[`enum NSVerticalDirection`](https://developer.apple.com/documentation/appkit/nsverticaldirection)
A direction on the vertical axis.
[`enum NSWritingToolsBehavior`](https://developer.apple.com/documentation/appkit/nswritingtoolsbehavior)
Constants that specify the Writing Tools experience for the underlying view.
[`struct NSWritingToolsResultOptions`](https://developer.apple.com/documentation/appkit/nswritingtoolsresultoptions)
Constants to specify what type of content to allow in Writing Tools suggestions or rewrites.
