# offset(z:)
Brings a view forward in Z by the provided distance in points.
visionOS 1.0+
```
nonisolated
func offset(z: [CGFloat](https://developer.apple.com/documentation/CoreFoundation/CGFloat-swift.struct)) -> some [View](https://developer.apple.com/documentation/swiftui/view)

```

## [Return Value](https://developer.apple.com/documentation/SwiftUI/View/offset\(z:\)#return-value)
A view that is extruded forward in Z by `distance`.
