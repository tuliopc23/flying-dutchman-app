# Text Display
Display text and check spelling.
## [Overview](https://developer.apple.com/documentation/appkit/text-display#overview)
In most cases, you can lay out your app’s text using the [`NSTextField`](https://developer.apple.com/documentation/appkit/nstextfield) or [`NSTextView`](https://developer.apple.com/documentation/appkit/nstextview) classes (or their subclasses). Use the [`NSTextField`](https://developer.apple.com/documentation/appkit/nstextfield) class to add either a label or a simple text input. Use the [`NSTextView`](https://developer.apple.com/documentation/appkit/nstextview) class to provide more comprehensive layout and editing features for larger bodies of text.
For example, [`NSTextView`](https://developer.apple.com/documentation/appkit/nstextview) supports rich text, attachments (graphics, file, and other), input management and key binding, and marked text attributes. [`NSTextView`](https://developer.apple.com/documentation/appkit/nstextview) works with the font panel and menu, rulers and paragraph styles, the Services facility (for example, the spell-checking service), and the pasteboard.
[`NSTextView`](https://developer.apple.com/documentation/appkit/nstextview) also allows customizing through delegation and notifications—you rarely need to subclass [`NSTextView`](https://developer.apple.com/documentation/appkit/nstextview). You rarely create instances of [`NSTextView`](https://developer.apple.com/documentation/appkit/nstextview) programmatically either, because objects on Interface Builder’s palettes, such as [`NSTextField`](https://developer.apple.com/documentation/appkit/nstextfield), [`NSForm`](https://developer.apple.com/documentation/appkit/nsform), and [`NSScrollView`](https://developer.apple.com/documentation/appkit/nsscrollview), already contain [`NSTextView`](https://developer.apple.com/documentation/appkit/nstextview) objects.
For even more powerful and more creative text manipulation (such as displaying text in a circle) see [TextKit](https://developer.apple.com/documentation/appkit/textkit).
### [Spell-checking](https://developer.apple.com/documentation/appkit/text-display#Spell-checking)
The [`NSSpellServer`](https://developer.apple.com/documentation/Foundation/NSSpellServer) class lets you define a spell-checking service and provide it as a service to other apps. To connect your app to a spell-checking service, use the [`NSSpellChecker`](https://developer.apple.com/documentation/appkit/nsspellchecker) class. The [`NSIgnoreMisspelledWords`](https://developer.apple.com/documentation/appkit/nsignoremisspelledwords) and [`NSChangeSpelling`](https://developer.apple.com/documentation/appkit/nschangespelling) protocols support the spell-checking mechanism.

A view that draws text and handles user interactions with that text.
