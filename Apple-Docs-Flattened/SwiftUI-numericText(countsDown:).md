# numericText(countsDown:)
Creates a content transition intended to be used with `Text` views displaying numeric text. In certain environments changes to the text will enable a nonstandard transition tailored to numeric characters that count up or down.
iOS 16.0+iPadOS 16.0+Mac Catalyst 16.0+macOS 13.0+tvOS 16.0+visionOS 1.0+watchOS 9.0+
```
static func numericText(countsDown: [Bool](https://developer.apple.com/documentation/Swift/Bool) = false) -> [ContentTransition](https://developer.apple.com/documentation/swiftui/contenttransition)
```

## [ Parameters ](https://developer.apple.com/documentation/SwiftUI/ContentTransition/numericText\(countsDown:\)#parameters) 

`countsDown`
    
True if the numbers represented by the text are counting downwards.
## [Return Value](https://developer.apple.com/documentation/SwiftUI/ContentTransition/numericText\(countsDown:\)#return-value)
A new content transition.
