# setWidth(_:forSegmentAt:)
Sets the width of the segment at the index you specify.
iOS 2.0+iPadOS 2.0+Mac Catalyst 13.1+tvOSvisionOS 1.0+
```
@[MainActor](https://developer.apple.com/documentation/Swift/MainActor)
func setWidth(
    _ width: [CGFloat](https://developer.apple.com/documentation/CoreFoundation/CGFloat-swift.struct),
    forSegmentAt segment: [Int](https://developer.apple.com/documentation/Swift/Int)
)
```

## [ Parameters ](https://developer.apple.com/documentation/uikit/uisegmentedcontrol/setwidth\(_:forsegmentat:\)#parameters) 

`width`
    
A float value specifying the width of the segment. The default value is {0.0}, which tells `UISegmentedControl` to automatically size the segment. 

`segment`
    
An index number identifying a segment in the control. It must be a number between 0 and the number of segments ([`numberOfSegments`](https://developer.apple.com/documentation/uikit/uisegmentedcontrol/numberofsegments)) minus 1; the segmented control pins values exceeding this upper range to the last segment.
