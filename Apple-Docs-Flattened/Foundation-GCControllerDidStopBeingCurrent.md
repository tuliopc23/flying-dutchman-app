# GCControllerDidStopBeingCurrent
A notification that posts when a controller stops being the current controller.
iOS 14.0+iPadOS 14.0+Mac Catalyst 14.0+macOS 11.0+tvOS 14.0+visionOS 1.0+
```
static let GCControllerDidStopBeingCurrent: [NSNotification](https://developer.apple.com/documentation/foundation/nsnotification).[Name](https://developer.apple.com/documentation/foundation/nsnotification/name-swift.struct)
```

## [Discussion](https://developer.apple.com/documentation/Foundation/NSNotification/Name-swift.struct/GCControllerDidStopBeingCurrent#Discussion)
The notification object is the [`GCController`](https://developer.apple.com/documentation/GameController/GCController) object that’s no longer current.
The system posts this notification on the main actor.
