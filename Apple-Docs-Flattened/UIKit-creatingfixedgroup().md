# creatingFixedGroup()
Places the item in a fixed group that a person can’t move or remove from the navigation bar during layout customization.
iOS 16.0+iPadOS 16.0+Mac Catalyst 16.0+visionOS 1.0+
```
@[MainActor](https://developer.apple.com/documentation/Swift/MainActor)
func creatingFixedGroup() -> [UIBarButtonItemGroup](https://developer.apple.com/documentation/uikit/uibarbuttonitemgroup)
```

## [Return Value](https://developer.apple.com/documentation/uikit/uibarbuttonitem/creatingfixedgroup\(\)#return-value)
A [`UIBarButtonItemGroup`](https://developer.apple.com/documentation/uikit/uibarbuttonitemgroup) that contains only this bar button item.
## [Discussion](https://developer.apple.com/documentation/uikit/uibarbuttonitem/creatingfixedgroup\(\)#Discussion)
A bar button item can only belong to one [`UIBarButtonItemGroup`](https://developer.apple.com/documentation/uikit/uibarbuttonitemgroup). If you add a bar button item to a new group, the system removes it from its previous group.
