Instance Method
# setDraggingSourceOperationMask(_:forLocal:)
Sets the default operation mask returned by `draggingSourceOperationMaskForLocal:` to `mask`.
macOS
```
@[MainActor](https://developer.apple.com/documentation/Swift/MainActor)
func setDraggingSourceOperationMask(
    _ mask: [NSDragOperation](https://developer.apple.com/documentation/appkit/nsdragoperation),
    forLocal isLocal: [Bool](https://developer.apple.com/documentation/Swift/Bool)
)
```

## [ Parameters ](https://developer.apple.com/documentation/appkit/nstableview/setdraggingsourceoperationmask\(_:forlocal:\)#parameters) 

`mask`
    
The drag operation mask. See [`NSDragOperation`](https://developer.apple.com/documentation/appkit/nsdragoperation) for the supported values. 

`isLocal`
    
[`true`](https://developer.apple.com/documentation/Swift/true) if the destination is the same application, otherwise [`false`](https://developer.apple.com/documentation/Swift/false). In either case the specified `mask` value is archived and used.
