# filter(_:)
Republishes all elements that match a provided closure.
iOS 13.0+iPadOS 13.0+Mac Catalyst 13.0+macOS 10.15+tvOS 13.0+visionOS 1.0+watchOS 6.0+
```
func filter(_ isIncluded: @escaping (Self.[Output](https://developer.apple.com/documentation/combine/publisher/output)) -> [Bool](https://developer.apple.com/documentation/Swift/Bool)) -> [Publishers](https://developer.apple.com/documentation/combine/publishers).[Filter](https://developer.apple.com/documentation/combine/publishers/filter)<Self>
```

## [ Parameters ](https://developer.apple.com/documentation/Combine/Publisher/filter\(_:\)#parameters) 

`isIncluded`
    
A closure that takes one element and returns a Boolean value indicating whether to republish the element.
## [Return Value](https://developer.apple.com/documentation/Combine/Publisher/filter\(_:\)#return-value)
A publisher that republishes all elements that satisfy the closure.

## [Discussion](https://developer.apple.com/documentation/Combine/Publisher/filter\(_:\)#discussion)
Combine’s [`filter(_:)`](https://developer.apple.com/documentation/combine/publisher/filter\(_:\)) operator performs an operation similar to that of [`filter(_:)`](https://developer.apple.com/documentation/Swift/Sequence/filter\(_:\)-5y9d2) in the Swift Standard Library: it uses a closure to test each element to determine whether to republish the element to the downstream subscriber.
The following example, uses a filter operation that receives an `Int` and only republishes a value if it’s even.
```
let numbers: [Int] = [1, 2, 3, 4, 5]
cancellable = numbers.publisher
    .filter { $0 % 2 == 0 }
    .sink { print("\($0)", terminator: " ") }


// Prints: "2 4"

```
