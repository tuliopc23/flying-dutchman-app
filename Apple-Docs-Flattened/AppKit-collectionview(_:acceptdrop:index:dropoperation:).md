# collectionView(_:acceptDrop:index:dropOperation:)
Invoked when the mouse is released over a collection view that previously allowed a drop.
macOS 10.6+
```
@[MainActor](https://developer.apple.com/documentation/Swift/MainActor)
optional func collectionView(
    _ collectionView: [NSCollectionView](https://developer.apple.com/documentation/appkit/nscollectionview),
    acceptDrop draggingInfo: any [NSDraggingInfo](https://developer.apple.com/documentation/appkit/nsdragginginfo),
    index: [Int](https://developer.apple.com/documentation/Swift/Int),
    dropOperation: [NSCollectionView](https://developer.apple.com/documentation/appkit/nscollectionview).[DropOperation](https://developer.apple.com/documentation/appkit/nscollectionview/dropoperation)
) -> [Bool](https://developer.apple.com/documentation/Swift/Bool)
```

## [ Parameters ](https://developer.apple.com/documentation/appkit/nscollectionviewdelegate/collectionview\(_:acceptdrop:index:dropoperation:\)#parameters) 

`collectionView`
    
The collection view that send the message. 

`draggingInfo`
    
An object that contains more information about this dragging operation. 

`index`
    
The index of the proposed drop item. 

`dropOperation`
    
The type of dragging operation.
## [Return Value](https://developer.apple.com/documentation/appkit/nscollectionviewdelegate/collectionview\(_:acceptdrop:index:dropoperation:\)#return-value)
[`true`](https://developer.apple.com/documentation/Swift/true) if the drop operation should be accepted, otherwise [`false`](https://developer.apple.com/documentation/Swift/false).
## [Discussion](https://developer.apple.com/documentation/appkit/nscollectionviewdelegate/collectionview\(_:acceptdrop:index:dropoperation:\)#Discussion)
This method is called when the mouse is released over a collection view that previously decided to allow a drop via the [`collectionView(_:validateDrop:proposedIndex:dropOperation:)`](https://developer.apple.com/documentation/appkit/nscollectionviewdelegate/collectionview\(_:validatedrop:proposedindex:dropoperation:\)) method. At this time, the delegate should incorporate the data from the dragging pasteboard and update the collection view’s contents.
You must implement this method for your collection view to be a drag destination
