# labelColor
The primary color to use for text labels.
macOS 10.10+
```
class var labelColor: [NSColor](https://developer.apple.com/documentation/appkit/nscolor) { get }
```

## [Discussion](https://developer.apple.com/documentation/AppKit/NSColor/labelColor#Discussion)
Use this color in the most important text labels of your user interface. You can also use it for other types of primary app content.
