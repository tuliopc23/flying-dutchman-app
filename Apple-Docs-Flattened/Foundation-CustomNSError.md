# CustomNSError
A specialized error that provides a domain, error code, and user-info dictionary.
iOS 8.0+iPadOS 8.0+Mac Catalyst 8.0+macOS 10.10+tvOS 9.0+visionOS 1.0+watchOS 2.0+
