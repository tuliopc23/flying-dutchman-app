Type Property
# automaticDimension
The default value to apply to a dimension.
macOS 10.11+
```
class let automaticDimension: [CGFloat](https://developer.apple.com/documentation/CoreFoundation/CGFloat-swift.struct)
```

## [Discussion](https://developer.apple.com/documentation/appkit/nssplitviewcontroller/automaticdimension#Discussion)
Use this constant with any sizing-related [`NSSplitViewController`](https://developer.apple.com/documentation/appkit/nssplitviewcontroller) property to get the default system behavior.
