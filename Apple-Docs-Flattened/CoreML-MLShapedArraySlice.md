# MLShapedArraySlice
A multidimensional subset of elements from a shaped array type.
iOS 15.0+iPadOS 15.0+Mac Catalyst 15.0+macOS 12.0+tvOS 15.0+visionOS 1.0+watchOS 8.0+
```
struct MLShapedArraySlice<Scalar> where Scalar : [MLShapedArrayScalar](https://developer.apple.com/documentation/coreml/mlshapedarrayscalar)
```
