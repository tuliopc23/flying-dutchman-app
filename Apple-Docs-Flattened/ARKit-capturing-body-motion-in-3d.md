# Capturing Body Motion in 3D
Track a person in the physical environment and visualize their motion by applying the same body movements to a virtual character.
[ Download ](https://docs-assets.developer.apple.com/published/78433fffa4d4/CapturingBodyMotionIn3D.zip)
iOS 13.0+iPadOS 13.0+Xcode 16.0+
## [Overview](https://developer.apple.com/documentation/arkit/capturing-body-motion-in-3d#Overview)
To run the sample, use an iOS device with an A12 chip or later.
Note
This sample code project is associated with WWDC 2019 session [607: Bringing People into AR](https://developer.apple.com/videos/play/wwdc19/607/).
