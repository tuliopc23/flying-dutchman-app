# Browser View
Provide a column-based interface for viewing and navigating hierarchical information.
## [Topics](https://developer.apple.com/documentation/appkit/browser-view#topics)
### [View](https://developer.apple.com/documentation/appkit/browser-view#View)
[`class NSBrowser`](https://developer.apple.com/documentation/appkit/nsbrowser)
An interface that displays a hierarchically organized list of data items that can be navigated and selected.
### [Management](https://developer.apple.com/documentation/appkit/browser-view#Management)
[`protocol NSBrowserDelegate`](https://developer.apple.com/documentation/appkit/nsbrowserdelegate)
A set of methods that a browser delegate implements to manage selection, scrolling, sizing, and other behavior.
### [Cell](https://developer.apple.com/documentation/appkit/browser-view#Cell)
[`class NSBrowserCell`](https://developer.apple.com/documentation/appkit/nsbrowsercell)
The user interface of a browser.
## [See Also](https://developer.apple.com/documentation/appkit/browser-view#see-also)
### [Content views](https://developer.apple.com/documentation/appkit/browser-view#Content-views)
[API Reference Collection View](https://developer.apple.com/documentation/appkit/collection-view)
Display one or more subviews in a highly configurable arrangement.
[API Reference Outline View](https://developer.apple.com/documentation/appkit/outline-view)
Display a list-based interface for hierarchical data, where each level of hierarchy is indented from the previous one.
[API Reference Table View](https://developer.apple.com/documentation/appkit/table-view)
Display custom data in rows and columns.
[`class NSTextView`](https://developer.apple.com/documentation/appkit/nstextview)
A view that draws text and handles user interactions with that text.
