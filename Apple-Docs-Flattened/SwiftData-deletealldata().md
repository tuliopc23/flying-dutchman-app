# deleteAllData()
Removes all persisted model data from the app’s persistent storage.
iOS 17.0–26.2DeprecatediPadOS 17.0–26.2DeprecatedMac Catalyst 17.0–26.2DeprecatedmacOS 14.0–26.2DeprecatedtvOS 17.0–26.2DeprecatedvisionOS 1.0–26.2DeprecatedwatchOS 10.0–26.2DeprecatedSwift 5.9+

## [Discussion](https://developer.apple.com/documentation/swiftdata/modelcontainer/deletealldata\(\)#discussion)
Warning
After you call this method, the container immediately deletes all data from the app’s persistent storage. This deletion is permanent and cannot be undone.
