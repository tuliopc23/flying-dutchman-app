# formatted(date:time:)
Generates a locale-aware string representation of a date using specified date and time format styles.
iOS 15.0+iPadOS 15.0+Mac Catalyst 15.0+macOS 12.0+tvOS 15.0+visionOS 1.0+watchOS 8.0+
```
func formatted(
    date: [Date](https://developer.apple.com/documentation/foundation/date).[FormatStyle](https://developer.apple.com/documentation/foundation/date/formatstyle).[DateStyle](https://developer.apple.com/documentation/foundation/date/formatstyle/datestyle),
    time: [Date](https://developer.apple.com/documentation/foundation/date).[FormatStyle](https://developer.apple.com/documentation/foundation/date/formatstyle).[TimeStyle](https://developer.apple.com/documentation/foundation/date/formatstyle/timestyle)
) -> [String](https://developer.apple.com/documentation/Swift/String)
```

## [ Parameters ](https://developer.apple.com/documentation/Foundation/Date/formatted\(date:time:\)#parameters) 

`date`
    
The date format style to apply to the date. 

`time`
    
The time format style to apply to the date.
## [Return Value](https://developer.apple.com/documentation/Foundation/Date/formatted\(date:time:\)#return-value)
A string, formatted according to the specified date and time styles.
## [Discussion](https://developer.apple.com/documentation/Foundation/Date/formatted\(date:time:\)#Discussion)
When displaying a date to a user, use the convenient [`formatted(date:time:)`](https://developer.apple.com/documentation/foundation/date/formatted\(date:time:\)) instance method to customize the string representation of the date. Set the date and time styles of the date format style separately, according to your particular needs.
For example, to create a string with a full date and no time representation, set the [`Date.FormatStyle.DateStyle`](https://developer.apple.com/documentation/foundation/date/formatstyle/datestyle) to [`complete`](https://developer.apple.com/documentation/foundation/date/formatstyle/datestyle/complete) and the [`Date.FormatStyle.TimeStyle`](https://developer.apple.com/documentation/foundation/date/formatstyle/timestyle) to [`omitted`](https://developer.apple.com/documentation/foundation/date/formatstyle/timestyle/omitted). Conversely, to create a string representing only the time, set the date style to [`omitted`](https://developer.apple.com/documentation/foundation/date/formatstyle/datestyle/omitted) and the time style to [`complete`](https://developer.apple.com/documentation/foundation/date/formatstyle/timestyle/complete).
```
let birthday = Date()


birthday.formatted(date: .complete, time: .omitted) // Sunday, January 17, 2021
birthday.formatted(date: .omitted, time: .complete) // 4:03:12 PM CST

```

You can create string representations of a [`Date`](https://developer.apple.com/documentation/foundation/date) instance with several levels of brevity using a variety of preset date and time styles. This example shows date styles of [`long`](https://developer.apple.com/documentation/foundation/date/formatstyle/datestyle/long), [`abbreviated`](https://developer.apple.com/documentation/foundation/date/formatstyle/datestyle/abbreviated), and [`numeric`](https://developer.apple.com/documentation/foundation/date/formatstyle/datestyle/numeric), and time styles of [`shortened`](https://developer.apple.com/documentation/foundation/date/formatstyle/timestyle/shortened), [`standard`](https://developer.apple.com/documentation/foundation/date/formatstyle/timestyle/standard), and [`complete`](https://developer.apple.com/documentation/foundation/date/formatstyle/timestyle/complete).
```
let birthday = Date()


birthday.formatted(date: .long, time: .shortened) // January 17, 2021, 4:03 PM
birthday.formatted(date: .abbreviated, time: .standard) // Jan 17, 2021, 4:03:12 PM
birthday.formatted(date: .numeric, time: .complete) // 1/17/2021, 4:03:12 PM CST


birthday.formatted() // Jan 17, 2021, 4:03 PM

```

The default date style is [`abbreviated`](https://developer.apple.com/documentation/foundation/date/formatstyle/datestyle/abbreviated) and the default time style is [`shortened`](https://developer.apple.com/documentation/foundation/date/formatstyle/timestyle/shortened).
For the default date formatting, use the [`formatted()`](https://developer.apple.com/documentation/foundation/date/formatted\(\)) method. To customize the formatted measurement string, use the [`formatted(_:)`](https://developer.apple.com/documentation/foundation/date/formatted\(_:\)) method and include a `Date.FormatStyle`.
For more information about formatting dates, see the [`Date.FormatStyle`](https://developer.apple.com/documentation/foundation/date/formatstyle).
