# DeferredProperty(title:)
A macro that creates an async property for an AppEntity that allows for providing an async get accessor.
iOS 26.0+iPadOS 26.0+Mac CatalystmacOS 26.0+tvOS 26.0+visionOS 26.0+watchOS 26.0+
```
@attached(peer, names: prefixed(`$`), prefixed(`_`)) @attached(accessor, names: named(get), named(set))
macro DeferredProperty(title: [LocalizedStringResource](https://developer.apple.com/documentation/Foundation/LocalizedStringResource))
```

## [ Parameters ](https://developer.apple.com/documentation/appintents/deferredproperty\(title:\)#parameters) 

`title`
    
A localized string resource describing the property for display in the user interface.
## [Overview](https://developer.apple.com/documentation/appintents/deferredproperty\(title:\)#overview)
Deferred properties have a few trade offs to consider:
  * They are not included when indexing and IndexedEntity
  * They are not sent to Shortcuts and Siri automatically and will only be fetched when needed


## [Example](https://developer.apple.com/documentation/appintents/deferredproperty\(title:\)#Example)
```
struct Restaurant: AppEntity {
    var model: Menu


    @DeferredProperty
    var popularItems: [MenuItem] {
        get async {
            await model.server.popularMenuItems()
        }
    }
}

```
