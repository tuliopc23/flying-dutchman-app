# collectionView(_:didSelectItemAt:)
Tells the delegate that the item at the specified index path was selected.
iOS 8.0+iPadOS 8.0+Mac Catalyst 13.1+tvOS 9.0+visionOS 1.0+
```
@[MainActor](https://developer.apple.com/documentation/Swift/MainActor)
optional func collectionView(
    _ collectionView: [UICollectionView](https://developer.apple.com/documentation/uikit/uicollectionview),
    didSelectItemAt indexPath: [IndexPath](https://developer.apple.com/documentation/Foundation/IndexPath)
)
```

## [ Parameters ](https://developer.apple.com/documentation/uikit/uicollectionviewdelegate/collectionview\(_:didselectitemat:\)#parameters) 

`collectionView`
    
The collection view object that is notifying you of the selection change. 

`indexPath`
    
The index path of the cell that was selected.
## [Discussion](https://developer.apple.com/documentation/uikit/uicollectionviewdelegate/collectionview\(_:didselectitemat:\)#Discussion)
The collection view calls this method when the user successfully selects an item in the collection view. It does not call this method when you programmatically set the selection.
