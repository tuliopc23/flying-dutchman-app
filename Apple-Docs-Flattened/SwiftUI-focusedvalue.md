# FocusedValue
A property wrapper for observing values from the focused view or one of its ancestors.
iOS 14.0+iPadOS 14.0+Mac Catalyst 14.0+macOS 11.0+tvOS 14.0+visionOS 1.0+watchOS 7.0+
```
@propertyWrapper
struct FocusedValue<Value>
```
