# primaryEdge
The side on which the primary view controller sits.
iOS 11.0+iPadOS 11.0+Mac Catalyst 13.1+tvOS 11.0+visionOS 1.0+
```
@[MainActor](https://developer.apple.com/documentation/Swift/MainActor)
var primaryEdge: [UISplitViewController](https://developer.apple.com/documentation/uikit/uisplitviewcontroller).[PrimaryEdge](https://developer.apple.com/documentation/uikit/uisplitviewcontroller/primaryedge-swift.enum) { get set }
```

## [Discussion](https://developer.apple.com/documentation/uikit/uisplitviewcontroller/primaryedge-swift.property#Discussion)
Use this property to change the default arrangement of the child view controllers. The default value of this property is [`UISplitViewController.PrimaryEdge.leading`](https://developer.apple.com/documentation/uikit/uisplitviewcontroller/primaryedge-swift.enum/leading).
