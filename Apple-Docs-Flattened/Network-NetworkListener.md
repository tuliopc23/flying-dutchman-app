# NetworkListener
Listen for incoming network connections.
iOS 26.0+iPadOS 26.0+Mac Catalyst 26.0+macOS 26.0+tvOS 26.0+visionOS 26.0+watchOS 26.0+
```
final class NetworkListener<ApplicationProtocol> where ApplicationProtocol : [NetworkProtocolOptions](https://developer.apple.com/documentation/network/networkprotocoloptions)
```

## [Overview](https://developer.apple.com/documentation/Network/NetworkListener#overview)
A listener receives incoming connections by binding to a local endpoint. It accepts connections based on the protocols defined in its protocol stack. Accepted connections will represent new local and remote address and port tuples.
