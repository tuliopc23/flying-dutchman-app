# Macros
Generate boilerplate code and perform other compile-time operations.
## [Topics](https://developer.apple.com/documentation/swift/macros#topics)
### [Essentials](https://developer.apple.com/documentation/swift/macros#Essentials)
[Applying Macros](https://developer.apple.com/documentation/swift/applying-macros)
Use macros to generate repetitive code at compile time.
### [Getting Source Location Information](https://developer.apple.com/documentation/swift/macros#Getting-Source-Location-Information)
[`macro file<T>() -> T`](https://developer.apple.com/documentation/swift/file\(\))
Produces the path to the file in which it appears.
[`macro fileID<T>() -> T`](https://developer.apple.com/documentation/swift/fileid\(\))
Produces a unique identifier for the source file in which the macro appears.
[`macro filePath<T>() -> T`](https://developer.apple.com/documentation/swift/filepath\(\))
Produces the complete path to the file in which the macro appears.
[`macro function<T>() -> T`](https://developer.apple.com/documentation/swift/function\(\))
Produces the name of the declaration in which it appears.
[`macro line<T>() -> T`](https://developer.apple.com/documentation/swift/line\(\))
Produces the line number on which it appears.
[`macro column<T>() -> T`](https://developer.apple.com/documentation/swift/column\(\))
Produces the column number in which the macro begins.
### [Generating Compile-Time Diagnostics](https://developer.apple.com/documentation/swift/macros#Generating-Compile-Time-Diagnostics)
[`macro warning(String)`](https://developer.apple.com/documentation/swift/warning\(_:\))
Produces the given warning message during compilation.
[`macro error(String)`](https://developer.apple.com/documentation/swift/error\(_:\))
Emits the given message as a fatal error and terminates the compilation process.
### [Writing Custom Macros](https://developer.apple.com/documentation/swift/macros#Writing-Custom-Macros)
[`macro externalMacro<T>(module: String, type: String) -> T`](https://developer.apple.com/documentation/swift/externalmacro\(module:type:\))
Specifies the module and type name for a macro’s implementation.
### [Accessing the Dynamic Shared Object Handle](https://developer.apple.com/documentation/swift/macros#Accessing-the-Dynamic-Shared-Object-Handle)
[`macro dsohandle() -> UnsafeRawPointer`](https://developer.apple.com/documentation/swift/dsohandle\(\))
Produces the dynamic shared object (DSO) handle in use where the macro appears.
## [See Also](https://developer.apple.com/documentation/swift/macros#see-also)
### [Programming Tasks](https://developer.apple.com/documentation/swift/macros#Programming-Tasks)
[API Reference Input and Output](https://developer.apple.com/documentation/swift/input-and-output)
Print values to the console, read from and write to text streams, and use command line arguments.
[API Reference Debugging and Reflection](https://developer.apple.com/documentation/swift/debugging-and-reflection)
Fortify your code with runtime checks, and examine your values’ runtime representation.
[API Reference Concurrency](https://developer.apple.com/documentation/swift/concurrency)
Perform asynchronous and parallel operations.
[API Reference Key-Path Expressions](https://developer.apple.com/documentation/swift/key-path-expressions)
Use key-path expressions to access properties dynamically.
[API Reference Manual Memory Management](https://developer.apple.com/documentation/swift/manual-memory-management)
Allocate and manage memory manually.
[API Reference Type Casting and Existential Types](https://developer.apple.com/documentation/swift/type-casting-and-existential-types)
Perform casts between types or represent values of any type.
[API Reference C Interoperability](https://developer.apple.com/documentation/swift/c-interoperability)
Use imported C types or call C variadic functions.
[Operator Declarations](https://developer.apple.com/documentation/swift/operator-declarations)
Work with prefix, postfix, and infix operators.
