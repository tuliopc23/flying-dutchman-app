# AVAudioEnvironmentDistanceAttenuationParameters
An object that specifies the amount of attenuation distance, the gradual loss in audio intensity, and other characteristics.
iOS 8.0+iPadOS 8.0+Mac Catalyst 13.1+macOS 10.10+tvOS 9.0+visionOS 1.0+watchOS 2.0+
```
class AVAudioEnvironmentDistanceAttenuationParameters
```

## [Overview](https://developer.apple.com/documentation/avfaudio/avaudioenvironmentdistanceattenuationparameters#overview)
Important
A source object (for example, [`AVAudioEnvironmentNode`](https://developer.apple.com/documentation/avfaudio/avaudioenvironmentnode)) provides an instance to this object. You can’t create standalone instances.

The distance attenuation model that describes the drop-off in gain as the source moves away from the listener.
[`enum AVAudioEnvironmentDistanceAttenuationModel`](https://developer.apple.com/documentation/avfaudio/avaudioenvironmentdistanceattenuationmodel)

### [Getting and Setting the Attenuation Values](https://developer.apple.com/documentation/avfaudio/avaudioenvironmentdistanceattenuationparameters#Getting-and-Setting-the-Attenuation-Values)
[`var maximumDistance: Float`](https://developer.apple.com/documentation/avfaudio/avaudioenvironmentdistanceattenuationparameters/maximumdistance)
The distance beyond which the node applies no further attenuation, in meters.
[`var referenceDistance: Float`](https://developer.apple.com/documentation/avfaudio/avaudioenvironmentdistanceattenuationparameters/referencedistance)
The minimum distance at which the node applies attenuation, in meters.
[`var rolloffFactor: Float`](https://developer.apple.com/documentation/avfaudio/avaudioenvironmentdistanceattenuationparameters/rollofffactor)
A factor that determines the attenuation curve.
