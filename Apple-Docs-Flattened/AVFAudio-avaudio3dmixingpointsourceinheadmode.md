# AVAudio3DMixingPointSourceInHeadMode
The in-head modes for a point source.
iOS 13.0+iPadOS 13.0+Mac Catalyst 13.1+macOS 10.15+tvOS 13.0+visionOS 1.0+
```
enum AVAudio3DMixingPointSourceInHeadMode
```

## [Overview](https://developer.apple.com/documentation/avfaudio/avaudio3dmixingpointsourceinheadmode#overview)
The in-head mode that determines what happens when a [`AVAudio3DMixingSourceMode.pointSource`](https://developer.apple.com/documentation/avfaudio/avaudio3dmixingsourcemode/pointsource) moves inside the head of the listener. The in-head mode applies when using the [`AVAudio3DMixingRenderingAlgorithm.auto`](https://developer.apple.com/documentation/avfaudio/avaudio3dmixingrenderingalgorithm/auto) rendering algorithm.
## [Topics](https://developer.apple.com/documentation/avfaudio/avaudio3dmixingpointsourceinheadmode#topics)
### [In-Head Modes](https://developer.apple.com/documentation/avfaudio/avaudio3dmixingpointsourceinheadmode#In-Head-Modes)
[`case mono`](https://developer.apple.com/documentation/avfaudio/avaudio3dmixingpointsourceinheadmode/mono)
The point source remains a single mono source inside the head of the listener regardless of the channels it consists of.
[`case bypass`](https://developer.apple.com/documentation/avfaudio/avaudio3dmixingpointsourceinheadmode/bypass)
The point source distributes into each output channel inside the head of the listener.
