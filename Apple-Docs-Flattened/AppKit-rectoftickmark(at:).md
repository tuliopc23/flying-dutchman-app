Instance Method
# rectOfTickMark(at:)
Returns the bounding rectangle of the tick mark at the given index.
macOS
```
@[MainActor](https://developer.apple.com/documentation/Swift/MainActor)
func rectOfTickMark(at index: [Int](https://developer.apple.com/documentation/Swift/Int)) -> [NSRect](https://developer.apple.com/documentation/Foundation/NSRect)
```

## [ Parameters ](https://developer.apple.com/documentation/appkit/nsslider/rectoftickmark\(at:\)#parameters) 

`index`
    
The index of the tick mark for which to retrieve the bounds. The minimum-value tick mark is at index `0`.
## [Return Value](https://developer.apple.com/documentation/appkit/nsslider/rectoftickmark\(at:\)#return-value)
The bounding rectangle of the specified tick mark.
## [Discussion](https://developer.apple.com/documentation/appkit/nsslider/rectoftickmark\(at:\)#Discussion)
If no tick mark is associated with `index`, the method raises `NSRangeException`. In its implementation of this method, the receiving `NSSlider` instance invokes the method of the same name of its [`NSSliderCell`](https://developer.apple.com/documentation/appkit/nsslidercell) instance.
