# View Management
Manage your user interface, including the size and position of views in a window.
## [Topics](https://developer.apple.com/documentation/appkit/view-management#topics)
### [Content Controllers](https://developer.apple.com/documentation/appkit/view-management#Content-Controllers)
[`class NSWindowController`](https://developer.apple.com/documentation/appkit/nswindowcontroller)
A controller that manages a window, usually a window stored in a nib file.
[`class NSViewController`](https://developer.apple.com/documentation/appkit/nsviewcontroller)
A controller that manages a view, typically loaded from a nib file.
[`class NSTitlebarAccessoryViewController`](https://developer.apple.com/documentation/appkit/nstitlebaraccessoryviewcontroller)
An object that manages a custom view—known as an accessory view—in the title bar–toolbar area of a window.
### [Split View Interface](https://developer.apple.com/documentation/appkit/view-management#Split-View-Interface)
[`class NSSplitViewController`](https://developer.apple.com/documentation/appkit/nssplitviewcontroller)
An object that manages an array of adjacent child views, and has a split view object for managing dividers between those views.
[`class NSSplitView`](https://developer.apple.com/documentation/appkit/nssplitview)
A view that arranges two or more views in a linear stack running horizontally or vertically.
[`class NSSplitViewItem`](https://developer.apple.com/documentation/appkit/nssplitviewitem)
An item in a split view controller.
### [Stack View Interface](https://developer.apple.com/documentation/appkit/view-management#Stack-View-Interface)
[`class NSStackView`](https://developer.apple.com/documentation/appkit/nsstackview)
A view that arranges an array of views horizontally or vertically and updates their placement and sizing when the window size changes.
### [Tab View Interface](https://developer.apple.com/documentation/appkit/view-management#Tab-View-Interface)
[`class NSTabViewController`](https://developer.apple.com/documentation/appkit/nstabviewcontroller)
A container view controller that manages a tab view interface, which organizes multiple pages of content but displays only one page at a time.
[`class NSTabView`](https://developer.apple.com/documentation/appkit/nstabview)
A multipage interface that displays one page at a time.
[`class NSTabViewItem`](https://developer.apple.com/documentation/appkit/nstabviewitem)
An item in a tab view.
### [Paged Interface](https://developer.apple.com/documentation/appkit/view-management#Paged-Interface)
[`class NSPageController`](https://developer.apple.com/documentation/appkit/nspagecontroller)
An object that controls swipe navigation and animations between views or view content.
### [Media Library Interface](https://developer.apple.com/documentation/appkit/view-management#Media-Library-Interface)
[`class NSMediaLibraryBrowserController`](https://developer.apple.com/documentation/appkit/nsmedialibrarybrowsercontroller)
An object that configures and displays a Media Library Browser panel.

Animate your views and other content to create a more engaging experience for users.
[API Reference Windows, Panels, and Screens](https://developer.apple.com/documentation/appkit/windows-panels-and-screens)
Organize your view hierarchies and facilitate their display onscreen.
[API Reference Sound, Speech, and Haptics](https://developer.apple.com/documentation/appkit/sound-speech-and-haptics)
Play sounds and haptic feedback, and incorporate speech recognition and synthesis into your interface.
