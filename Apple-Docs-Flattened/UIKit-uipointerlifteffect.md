# UIPointerLiftEffect
An effect where the pointer slides under the given view and disappears as the view scales up and gains a shadow.
iOS 13.4+iPadOS 13.4+Mac Catalyst 13.4+visionOS 1.0+
```
@interface UIPointerLiftEffect : [UIPointerEffect](https://developer.apple.com/documentation/uikit/uipointereffect-c.class)
```
