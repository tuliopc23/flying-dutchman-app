# onDelete(perform:)
Sets the deletion action for the dynamic view. You must delete the corresponding item within `action`, as it will be called after the row has already been removed from the [`List`](https://developer.apple.com/documentation/swiftui/list).
iOS 13.0+iPadOS 13.0+Mac Catalyst 13.0+macOS 10.15+tvOS 13.0+visionOS 1.0+watchOS 6.0+
```
nonisolated
func onDelete(perform action: [Optional](https://developer.apple.com/documentation/Swift/Optional)<([IndexSet](https://developer.apple.com/documentation/Foundation/IndexSet)) -> [Void](https://developer.apple.com/documentation/Swift/Void)>) -> some [DynamicViewContent](https://developer.apple.com/documentation/swiftui/dynamicviewcontent)

```

## [ Parameters ](https://developer.apple.com/documentation/SwiftUI/DynamicViewContent/onDelete\(perform:\)#parameters) 

`action`
    
The action that you want SwiftUI to perform when elements in the view are deleted. SwiftUI passes a set of indices to the closure that’s relative to the dynamic view’s underlying collection of data.
## [Return Value](https://developer.apple.com/documentation/SwiftUI/DynamicViewContent/onDelete\(perform:\)#return-value)
A view that calls `action` when elements are deleted from the original view.
