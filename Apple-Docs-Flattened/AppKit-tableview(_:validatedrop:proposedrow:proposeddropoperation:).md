# tableView(_:validateDrop:proposedRow:proposedDropOperation:)
Used by `aTableView` to determine a valid drop target.
macOS
```
@[MainActor](https://developer.apple.com/documentation/Swift/MainActor)
optional func tableView(
    _ tableView: [NSTableView](https://developer.apple.com/documentation/appkit/nstableview),
    validateDrop info: any [NSDraggingInfo](https://developer.apple.com/documentation/appkit/nsdragginginfo),
    proposedRow row: [Int](https://developer.apple.com/documentation/Swift/Int),
    proposedDropOperation dropOperation: [NSTableView](https://developer.apple.com/documentation/appkit/nstableview).[DropOperation](https://developer.apple.com/documentation/appkit/nstableview/dropoperation)
) -> [NSDragOperation](https://developer.apple.com/documentation/appkit/nsdragoperation)
```

## [ Parameters ](https://developer.apple.com/documentation/appkit/nstableviewdatasource/tableview\(_:validatedrop:proposedrow:proposeddropoperation:\)#parameters) 

`tableView`
    
The table view that sent the message. 

`info`
    
An object that contains more information about this dragging operation. 

`row`
    
The index of the proposed target row. 

`dropOperation`
    
The type of dragging operation proposed.
## [Return Value](https://developer.apple.com/documentation/appkit/nstableviewdatasource/tableview\(_:validatedrop:proposedrow:proposeddropoperation:\)#return-value)
The dragging operation the data source will perform.
## [Discussion](https://developer.apple.com/documentation/appkit/nstableviewdatasource/tableview\(_:validatedrop:proposedrow:proposeddropoperation:\)#Discussion)
The data source may retarget a drop by calling [`setDropRow(_:dropOperation:)`](https://developer.apple.com/documentation/appkit/nstableview/setdroprow\(_:dropoperation:\)) and returning something other than `NSDragOperationNone`. A data source might retarget for various reasons, such as to provide better visual feedback when inserting into a sorted position.
To propose a drop on the second row, `row` would be 2 and `operation` would be `NSTableViewDropOn`. To propose a drop below the last row, `row` would be `[aTableView numberOfRows]` and `operation` would be `NSTableViewDropAbove`.
