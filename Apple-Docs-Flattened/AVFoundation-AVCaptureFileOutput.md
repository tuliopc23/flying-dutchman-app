# AVCaptureFileOutput
The abstract superclass for capture outputs that can record captured data to a file.
iOS 4.0+iPadOS 4.0+Mac Catalyst 14.0+macOS 10.7+tvOS 17.0+
```
class AVCaptureFileOutput
```

## [Topics](https://developer.apple.com/documentation/avfoundation/avcapturefileoutput#topics)
### [Setting file output properties](https://developer.apple.com/documentation/avfoundation/avcapturefileoutput#Setting-file-output-properties)
[`var delegate: (any AVCaptureFileOutputDelegate)?`](https://developer.apple.com/documentation/avfoundation/avcapturefileoutput/delegate)
The delegate object for the capture file output.
[`var maxRecordedDuration: CMTime`](https://developer.apple.com/documentation/avfoundation/avcapturefileoutput/maxrecordedduration)
The longest duration allowed for the recording.
[`var maxRecordedFileSize: Int64`](https://developer.apple.com/documentation/avfoundation/avcapturefileoutput/maxrecordedfilesize)
The maximum size, in bytes, of the data that should be recorded by the receiver.
[`var minFreeDiskSpaceLimit: Int64`](https://developer.apple.com/documentation/avfoundation/avcapturefileoutput/minfreediskspacelimit)
The minimum amount of free space, in bytes, required for recording to continue on a given volume.
[`var outputFileURL: URL?`](https://developer.apple.com/documentation/avfoundation/avcapturefileoutput/outputfileurl)
The URL to which output is directed.
[`var recordedDuration: CMTime`](https://developer.apple.com/documentation/avfoundation/avcapturefileoutput/recordedduration)
Indicates the duration of the media recorded to the current output file.
[`var recordedFileSize: Int64`](https://developer.apple.com/documentation/avfoundation/avcapturefileoutput/recordedfilesize)
Indicates the size, in bytes, of the data recorded to the current output file.
[`var isRecording: Bool`](https://developer.apple.com/documentation/avfoundation/avcapturefileoutput/isrecording)
Indicates whether recording is in progress.
[`var isRecordingPaused: Bool`](https://developer.apple.com/documentation/avfoundation/avcapturefileoutput/isrecordingpaused)
Indicates whether recording to the current output file is paused.
