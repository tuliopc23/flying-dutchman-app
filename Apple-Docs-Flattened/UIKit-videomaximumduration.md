Instance Property
# videoMaximumDuration
The maximum duration, in seconds, for a video recording.
iOS 3.1+iPadOS 3.1+Mac Catalyst 13.1+
```
@[MainActor](https://developer.apple.com/documentation/Swift/MainActor)
var videoMaximumDuration: [TimeInterval](https://developer.apple.com/documentation/Foundation/TimeInterval) { get set }
```

## [Discussion](https://developer.apple.com/documentation/uikit/uiimagepickercontroller/videomaximumduration#Discussion)
The default value for this property is 10 minutes (600 seconds). When a user taps the Share button to send a movie to MMS, MobileMe, YouTube, or another destination, an appropriate duration limit and an appropriate video quality are enforced.
This property is available only if the [`mediaTypes`](https://developer.apple.com/documentation/uikit/uiimagepickercontroller/mediatypes) property’s value array includes the `kUTTypeMovie` media type.
