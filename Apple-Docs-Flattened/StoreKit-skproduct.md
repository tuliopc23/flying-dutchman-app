[`SKProduct`](https://developer.apple.com/documentation/storekit/skproduct) objects are returned as part of an [`SKProductsResponse`](https://developer.apple.com/documentation/storekit/skproductsresponse) object.
## [Topics](https://developer.apple.com/documentation/storekit/skproduct#topics)
### [Getting the Product Identifier](https://developer.apple.com/documentation/storekit/skproduct#Getting-the-Product-Identifier)
[`var productIdentifier: String`](https://developer.apple.com/documentation/storekit/skproduct/productidentifier)
The string that identifies the product to the Apple App Store.
### [Getting Product Attributes](https://developer.apple.com/documentation/storekit/skproduct#Getting-Product-Attributes)
[`var localizedDescription: String`](https://developer.apple.com/documentation/storekit/skproduct/localizeddescription)
A description of the product.
[`var localizedTitle: String`](https://developer.apple.com/documentation/storekit/skproduct/localizedtitle)
The name of the product.
[`var contentVersion: String`](https://developer.apple.com/documentation/storekit/skproduct/contentversion)
A string that identifies the version of the content.
[`var isFamilyShareable: Bool`](https://developer.apple.com/documentation/storekit/skproduct/isfamilyshareable)
A Boolean value that indicates whether the product is available for Family Sharing in App Store Connect.
[`var contentLengths: [NSNumber]`](https://developer.apple.com/documentation/storekit/skproduct/contentlengths)
The total size of the content, in bytes.
### [Getting Pricing Information](https://developer.apple.com/documentation/storekit/skproduct#Getting-Pricing-Information)
[`var price: NSDecimalNumber`](https://developer.apple.com/documentation/storekit/skproduct/price)
The cost of the product in the local currency.
[`var priceLocale: Locale`](https://developer.apple.com/documentation/storekit/skproduct/pricelocale)
The locale used to format the price of the product.
[`var introductoryPrice: SKProductDiscount?`](https://developer.apple.com/documentation/storekit/skproduct/introductoryprice)
The object containing introductory price information for the product.
[`var discounts: [SKProductDiscount]`](https://developer.apple.com/documentation/storekit/skproduct/discounts)
An array of subscription offers available for the auto-renewable subscription.
[`class SKProductDiscount`](https://developer.apple.com/documentation/storekit/skproductdiscount)
The details of an introductory offer or a promotional offer for an auto-renewable subscription.
### [Getting Subscription Information](https://developer.apple.com/documentation/storekit/skproduct#Getting-Subscription-Information)
[`var subscriptionGroupIdentifier: String?`](https://developer.apple.com/documentation/storekit/skproduct/subscriptiongroupidentifier)
The identifier of the subscription group to which the subscription belongs.
[`var subscriptionPeriod: SKProductSubscriptionPeriod?`](https://developer.apple.com/documentation/storekit/skproduct/subscriptionperiod)
The period details for products that are subscriptions.
[`class SKProductSubscriptionPeriod`](https://developer.apple.com/documentation/storekit/skproductsubscriptionperiod)
An object containing the subscription period duration information.
[`enum PeriodUnit`](https://developer.apple.com/documentation/storekit/skproduct/periodunit)
Values representing the duration of an interval, from a day up to a year.
### [Getting Downloadable Content Information](https://developer.apple.com/documentation/storekit/skproduct#Getting-Downloadable-Content-Information)
[`var isDownloadable: Bool`](https://developer.apple.com/documentation/storekit/skproduct/isdownloadable)
A Boolean value that indicates whether the App Store has downloadable content for this product.
[`var downloadContentLengths: [NSNumber]`](https://developer.apple.com/documentation/storekit/skproduct/downloadcontentlengths)
The lengths of the downloadable files available for this product.
[`var downloadContentVersion: String`](https://developer.apple.com/documentation/storekit/skproduct/downloadcontentversion)
A string that identifies which version of the content is available for download.
[`var downloadable: Bool`](https://developer.apple.com/documentation/storekit/skproduct/downloadable)
A Boolean value that indicates whether the App Store has downloadable content for this product.
