Instance Method
# backButtonBackgroundImage(for:barMetrics:)
Returns the back button background image for a specified control state and bar metrics.
iOS 5.0+iPadOS 5.0+Mac Catalyst 13.1+visionOS 1.0+
```
@[MainActor](https://developer.apple.com/documentation/Swift/MainActor)
func backButtonBackgroundImage(
    for state: [UIControl](https://developer.apple.com/documentation/uikit/uicontrol).[State](https://developer.apple.com/documentation/uikit/uicontrol/state-swift.struct),
    barMetrics: [UIBarMetrics](https://developer.apple.com/documentation/uikit/uibarmetrics)
) -> [UIImage](https://developer.apple.com/documentation/uikit/uiimage)?
```

## [ Parameters ](https://developer.apple.com/documentation/uikit/uibarbuttonitem/backbuttonbackgroundimage\(for:barmetrics:\)#parameters) 

`state`
    
A control state. 

`barMetrics`
    
Bar metrics.
## [Return Value](https://developer.apple.com/documentation/uikit/uibarbuttonitem/backbuttonbackgroundimage\(for:barmetrics:\)#return-value)
The back button background image for `state` and `barMetrics`.
## [Discussion](https://developer.apple.com/documentation/uikit/uibarbuttonitem/backbuttonbackgroundimage\(for:barmetrics:\)#Discussion)
This modifier applies only to navigation bar back buttons and is ignored by other buttons.
