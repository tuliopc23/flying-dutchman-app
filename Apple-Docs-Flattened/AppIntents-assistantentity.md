# AssistantEntity
An app entity that Siri can access to fulfill a person’s request.
iOS 18.0+iPadOS 18.0+Mac CatalystmacOS 15.0+tvOS 18.0+visionOS 2.0+watchOS 11.0+
```
protocol AssistantEntity : [AppEntity](https://developer.apple.com/documentation/appintents/appentity)
```

## [Overview](https://developer.apple.com/documentation/appintents/assistantentity#overview)
Don’t adopt this protocol directly, instead use the [`AssistantEntity(schema:)`](https://developer.apple.com/documentation/appintents/assistantentity\(schema:\)) macro to meet requirements for making your [`AppEntity`](https://developer.apple.com/documentation/appintents/appentity) available to Siri.
