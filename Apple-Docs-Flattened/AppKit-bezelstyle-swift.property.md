# bezelStyle
The appearance of the button’s border.
macOS
```
@[MainActor](https://developer.apple.com/documentation/Swift/MainActor)
var bezelStyle: [NSButton](https://developer.apple.com/documentation/appkit/nsbutton).[BezelStyle](https://developer.apple.com/documentation/appkit/nsbutton/bezelstyle-swift.enum) { get set }
```

## [Return Value](https://developer.apple.com/documentation/appkit/nsbutton/bezelstyle-swift.property#return-value)
The bezel style of the button. See [`NSButton.BezelStyle`](https://developer.apple.com/documentation/appkit/nsbutton/bezelstyle-swift.enum) in [`NSButtonCell`](https://developer.apple.com/documentation/appkit/nsbuttoncell) for the list of possible values.
## [Discussion](https://developer.apple.com/documentation/appkit/nsbutton/bezelstyle-swift.property#discussion)
Note that if the button is not bordered, the bezel style is ignored.
