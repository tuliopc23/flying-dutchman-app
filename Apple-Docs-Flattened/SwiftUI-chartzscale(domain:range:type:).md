# chartZScale(domain:range:type:)
Configures the z scale for 3D charts.
iOS 26.0+iPadOS 26.0+Mac Catalyst 26.0+macOS 26.0+visionOS 26.0+
```
nonisolated
func chartZScale<Domain, Range>(
    domain: Domain,
    range: Range,
    type: [ScaleType](https://developer.apple.com/documentation/Charts/ScaleType)? = nil
) -> some [View](https://developer.apple.com/documentation/swiftui/view) where Domain : [ScaleDomain](https://developer.apple.com/documentation/Charts/ScaleDomain), Range : [PositionScaleRange](https://developer.apple.com/documentation/Charts/PositionScaleRange)

```

## [ Parameters ](https://developer.apple.com/documentation/swiftui/view/chartzscale\(domain:range:type:\)#parameters) 

`domain`
    
The possible data values along the z axis in the chart. You can define the domain with a `ClosedRange` for numeric values. 

`range`
    
The range of x positions that correspond to the scale domain. By default the range is determined by the dimension of the plot area. 

`type`
    
The scale type.
