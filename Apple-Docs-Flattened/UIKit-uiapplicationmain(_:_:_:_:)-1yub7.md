# UIApplicationMain(_:_:_:_:)
Creates the application object and the application delegate and sets up the event cycle.
iOSiPadOSMac CatalysttvOSvisionOS
```
func UIApplicationMain(
    _ argc: [Int32](https://developer.apple.com/documentation/Swift/Int32),
    _ argv: [UnsafeMutablePointer](https://developer.apple.com/documentation/Swift/UnsafeMutablePointer)<[UnsafeMutablePointer](https://developer.apple.com/documentation/Swift/UnsafeMutablePointer)<[CChar](https://developer.apple.com/documentation/Swift/CChar)>?>,
    _ principalClassName: [String](https://developer.apple.com/documentation/Swift/String)?,
    _ delegateClassName: [String](https://developer.apple.com/documentation/Swift/String)?
) -> [Int32](https://developer.apple.com/documentation/Swift/Int32)
```

## [ Parameters ](https://developer.apple.com/documentation/uikit/uiapplicationmain\(_:_:_:_:\)-1yub7#parameters) 

`argc`
    
The count of arguments in `argv`; this usually is the corresponding parameter to `main`. 

`argv`
    
A variable list of arguments; this usually is the corresponding parameter to `main`. 

`principalClassName`
    
The name of the [`UIApplication`](https://developer.apple.com/documentation/uikit/uiapplication) class or subclass. If you specify `nil`, `UIApplication` is assumed. 

`delegateClassName`
    
The name of the class from which the application delegate is instantiated. If `principalClassName` designates a subclass of [`UIApplication`](https://developer.apple.com/documentation/uikit/uiapplication), you may designate the subclass as the delegate; the subclass instance receives the application-delegate messages. Specify `nil` if you load the delegate object from your application’s main nib file.
## [Return Value](https://developer.apple.com/documentation/uikit/uiapplicationmain\(_:_:_:_:\)-1yub7#return-value)
Even though an integer return type is specified, this function never returns. When users exits an iOS app by pressing the Home button, the application moves to the background.

## [Discussion](https://developer.apple.com/documentation/uikit/uiapplicationmain\(_:_:_:_:\)-1yub7#Discussion)
This function instantiates the application object from the principal class and instantiates the delegate (if any) from the given class and sets the delegate for the application. It also sets up the main event loop, including the application’s run loop, and begins processing events. If the application’s `Info.plist` file specifies a main nib file to be loaded, by including the [`NSMainNibFile`](https://developer.apple.com/documentation/BundleResources/Information-Property-List/NSMainNibFile) key and a valid nib file name for the value, this function loads that nib file.
Despite the declared return type, this function never returns.
