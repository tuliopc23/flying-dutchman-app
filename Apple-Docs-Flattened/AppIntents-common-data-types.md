# Common types
Specify common types that your app supports, including currencies, files, and contacts.
## [Overview](https://developer.apple.com/documentation/appintents/common-data-types#Overview)
Use these types to manage specific types of data when you create a parameter for an app intent or a property for an app entity.
## [Topics](https://developer.apple.com/documentation/appintents/common-data-types#topics)
### [Contacts](https://developer.apple.com/documentation/appintents/common-data-types#Contacts)
[`struct IntentPerson`](https://developer.apple.com/documentation/appintents/intentperson)
Information that identifies a person participating in an intents-based interaction.
### [Files](https://developer.apple.com/documentation/appintents/common-data-types#Files)
[`struct IntentFile`](https://developer.apple.com/documentation/appintents/intentfile)
An interface for providing an app entity that represents an on-disk file or file-based resource.
### [Monetary types](https://developer.apple.com/documentation/appintents/common-data-types#Monetary-types)
[`struct IntentCurrencyAmount`](https://developer.apple.com/documentation/appintents/intentcurrencyamount)
An amount of money to transfer during a financial transaction.
[`struct IntentPaymentMethod`](https://developer.apple.com/documentation/appintents/intentpaymentmethod)
Information about a form of payment supported by your app.
### [Items and collections](https://developer.apple.com/documentation/appintents/common-data-types#Items-and-collections)
[`struct IntentItem`](https://developer.apple.com/documentation/appintents/intentitem)
A type describing a value returned from a dynamic options provider, plus information about how to display it to users.
[`struct IntentItemCollection`](https://developer.apple.com/documentation/appintents/intentitemcollection)
Return this object to provide an advanced list of options, optionally divided in sections.
[`struct IntentItemSection`](https://developer.apple.com/documentation/appintents/intentitemsection)
An object you use to divide dynamic options into sections.
[`struct IntentCollectionSize`](https://developer.apple.com/documentation/appintents/intentcollectionsize)
