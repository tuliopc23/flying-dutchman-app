Function
# pointwiseMin(_:_:)
Computes the element-wise minimum of two tensors.
iOS 18.0+iPadOS 18.0+Mac Catalyst 18.0+macOS 15.0+tvOS 18.0+visionOS 2.0+watchOS 11.0+
```
func pointwiseMin(
    _ lhs: [MLTensor](https://developer.apple.com/documentation/coreml/mltensor),
    _ rhs: [MLTensor](https://developer.apple.com/documentation/coreml/mltensor)
) -> [MLTensor](https://developer.apple.com/documentation/coreml/mltensor)
```

Show all declarations 
## [Discussion](https://developer.apple.com/documentation/coreml/pointwisemin\(_:_:\)#discussion)
For example:
```
let x = MLTensor([1.0, 3.0, 6.0])
let y = MLTensor([6.0, 3.0, 1.0])
let z = pointwiseMin(x, y)
await z.shapedArray(of: Float.self) // is [1.0, 3.0, 1.0]

```

Shapes must be broadcastable, where the broadcasted shape becomes the shape of the output.
