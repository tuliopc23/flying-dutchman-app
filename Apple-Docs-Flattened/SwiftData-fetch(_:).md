# fetch(_:)
Returns an array of typed models that match the criteria of the specified fetch descriptor.
iOS 17.0+iPadOS 17.0+Mac Catalyst 17.0+macOS 14.0+tvOS 17.0+visionOS 1.0+watchOS 10.0+Swift 5.9+
```
func fetch<T>(_ descriptor: [FetchDescriptor](https://developer.apple.com/documentation/swiftdata/fetchdescriptor)<T>) throws -> [T] where T : [PersistentModel](https://developer.apple.com/documentation/swiftdata/persistentmodel)
```

## [ Parameters ](https://developer.apple.com/documentation/SwiftData/ModelContext/fetch\(_:\)#parameters) 

`descriptor`
    
A fetch descriptor that provides the configuration for the fetch.
## [Return Value](https://developer.apple.com/documentation/SwiftData/ModelContext/fetch\(_:\)#return-value)
The array of typed models that satisfy the criteria of the fetch descriptor. If no models match the criteria, the array is empty.
