# Morphology.Pronoun
A custom pronoun for referring to a third person.
iOS 17.0+iPadOS 17.0+Mac Catalyst 17.0+macOS 14.0+tvOS 17.0+visionOS 1.0+watchOS 10.0+
```
struct Pronoun
```

## [Overview](https://developer.apple.com/documentation/foundation/morphology/pronoun#overview)
Create instances of [`Morphology.Pronoun`](https://developer.apple.com/documentation/foundation/morphology/pronoun) when you need to define custom pronouns for a localized term of address.
For examples of how to create custom pronouns, see [`TermOfAddress`](https://developer.apple.com/documentation/foundation/termofaddress).
