# transformed(by:)
Returns a new image that represents the original image after applying an affine transform.
iOS 5.0+iPadOS 5.0+Mac Catalyst 13.1+macOS 10.4+tvOSvisionOS 1.0+
```
func transformed(by matrix: [CGAffineTransform](https://developer.apple.com/documentation/CoreFoundation/CGAffineTransform)) -> [CIImage](https://developer.apple.com/documentation/coreimage/ciimage)
```

`matrix`
    
An affine transform.
