Initializer
# init(data:encoding:)
Returns a `String` initialized by converting given `data` into Unicode characters using a given `encoding`.
iOS 8.0+iPadOS 8.0+Mac Catalyst 8.0+macOS 10.10+tvOS 9.0+visionOS 1.0+watchOS 2.0+
```
init?(
    data: [Data](https://developer.apple.com/documentation/Foundation/Data),
    encoding: [String](https://developer.apple.com/documentation/swift/string).[Encoding](https://developer.apple.com/documentation/swift/string/encoding)
)
```
