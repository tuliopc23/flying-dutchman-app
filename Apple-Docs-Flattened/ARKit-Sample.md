Structure
# CameraFrame.Sample
Information that describes a sample from a camera frame.
visionOS 2.0+
```
struct Sample
```

## [Overview](https://developer.apple.com/documentation/ARKit/CameraFrame/Sample#overview)
Use this structure to access information about a sample from a [`CameraFrame`](https://developer.apple.com/documentation/arkit/cameraframe), such as the frame’s settings and parameters, a pixel buffer that contains the sample’s data, and so on.
## [Topics](https://developer.apple.com/documentation/ARKit/CameraFrame/Sample#topics)

The frame’s parameters.
[`struct Parameters`](https://developer.apple.com/documentation/arkit/cameraframe/sample/parameters-swift.struct)
A frame’s parameters, such as the camera type, intrinsics, timestamps, exposure, and so on.
