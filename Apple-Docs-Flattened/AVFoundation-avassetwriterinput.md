# AVAssetWriterInput
An object that appends media samples to a track in an asset writer’s output file.
iOS 4.1+iPadOS 4.1+Mac Catalyst 13.1+macOS 10.7+tvOS 9.0+visionOS 1.0+
```
class AVAssetWriterInput
```

## [ Mentioned in ](https://developer.apple.com/documentation/avfoundation/avassetwriterinput#mentions)
[](https://developer.apple.com/documentation/avfoundation/tagging-media-with-video-color-information)
## [Overview](https://developer.apple.com/documentation/avfoundation/avassetwriterinput#overview)
Create an asset writer input to write a single track of media, and optional track-level metadata, to the output file. To write multiple concurrent tracks with ideal interleaving of media data, observe the value of the [`isReadyForMoreMediaData`](https://developer.apple.com/documentation/avfoundation/avassetwriterinput/isreadyformoremediadata) property of each input.
You can use an asset writer input to create tracks in a QuickTime movie file that aren’t self-contained, and instead reference sample data that exists in another file.
