# NSPickerTouchBarItem.ControlRepresentation
Constants that specify display styles for picker bar items.
iOS 13.0+iPadOS 13.0+Mac Catalyst 13.1+macOS 10.15+
```
enum ControlRepresentation
```

[`case automatic`](https://developer.apple.com/documentation/appkit/nspickertouchbaritem/controlrepresentation-swift.enum/automatic)
The system dynamically changes the display mode based on the available space.
[`case expanded`](https://developer.apple.com/documentation/appkit/nspickertouchbaritem/controlrepresentation-swift.enum/expanded)
The system displays the control and all of its options directly.
[`case collapsed`](https://developer.apple.com/documentation/appkit/nspickertouchbaritem/controlrepresentation-swift.enum/collapsed)
The system displays the control’s options through a popover.
