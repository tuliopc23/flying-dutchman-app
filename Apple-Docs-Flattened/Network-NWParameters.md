# NWParameters
An object that stores the protocols to use for connections, options for sending data, and network path constraints.
iOS 12.0+iPadOS 12.0+Mac Catalyst 12.0+macOS 10.14+tvOS 12.0+visionOS 1.0+watchOS 5.0+
