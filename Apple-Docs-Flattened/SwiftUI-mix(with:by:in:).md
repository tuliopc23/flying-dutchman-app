# mix(with:by:in:)
Returns a version of self mixed with `rhs` by the amount specified by `fraction`.
iOS 18.0+iPadOS 18.0+Mac Catalyst 18.0+macOS 15.0+tvOS 18.0+visionOS 2.0+watchOS 11.0+
```
func mix(
    with rhs: [Color](https://developer.apple.com/documentation/swiftui/color),
    by fraction: [Double](https://developer.apple.com/documentation/Swift/Double),
    in colorSpace: [Gradient](https://developer.apple.com/documentation/swiftui/gradient).[ColorSpace](https://developer.apple.com/documentation/swiftui/gradient/colorspace) = .perceptual
) -> [Color](https://developer.apple.com/documentation/swiftui/color)
```

## [ Parameters ](https://developer.apple.com/documentation/SwiftUI/Color/mix\(with:by:in:\)#parameters) 

`rhs`
    
The color to mix `self` with. 

`fraction`
    
The amount of blending, `0.5` means `self` is mixed in equal parts with `rhs`. 

`colorSpace`
    
The color space used to mix the colors.
## [Return Value](https://developer.apple.com/documentation/SwiftUI/Color/mix\(with:by:in:\)#return-value)
A new `Color` based on `self` and `rhs`.
