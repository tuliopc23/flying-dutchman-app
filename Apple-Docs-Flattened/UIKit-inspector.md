# UISplitViewController.Column.inspector
The column for the inspector, or trailing, view controller.
iOS 26.0+iPadOS 26.0+Mac Catalyst 26.0+
```
case inspector
```

## [Overview](https://developer.apple.com/documentation/UIKit/UISplitViewController/Column/inspector#Overview)
By default, the split view controller hides the `inspector` column. To show the `inspector` column, call [`show(_:)`](https://developer.apple.com/documentation/uikit/uisplitviewcontroller/show\(_:\)).
