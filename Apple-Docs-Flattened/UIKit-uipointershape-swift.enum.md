# UIPointerShape
An object that defines the shape of custom pointers.
iOS 13.4+iPadOS 13.4+Mac CatalystvisionOS
```
enum UIPointerShape
```

## [Overview](https://developer.apple.com/documentation/uikit/uipointershape-swift.enum#overview)
If the desired pointer shape can be expressed as a rounded rectangle, use [`UIPointerShape.roundedRect(_:radius:)`](https://developer.apple.com/documentation/uikit/uipointershape-swift.enum/roundedrect\(_:radius:\)) for best results.
## [Topics](https://developer.apple.com/documentation/uikit/uipointershape-swift.enum#topics)
### [Specifying pointer shapes](https://developer.apple.com/documentation/uikit/uipointershape-swift.enum#Specifying-pointer-shapes)
[`case horizontalBeam(length: CGFloat)`](https://developer.apple.com/documentation/uikit/uipointershape-swift.enum/horizontalbeam\(length:\))
The pointer morphs into a horizontal beam using the specified length.
[`case verticalBeam(length: CGFloat)`](https://developer.apple.com/documentation/uikit/uipointershape-swift.enum/verticalbeam\(length:\))
The pointer morphs into a vertical beam using the specified length.
[`case path(UIBezierPath)`](https://developer.apple.com/documentation/uikit/uipointershape-swift.enum/path\(_:\))
The pointer morphs into the given Bézier path.
[`case roundedRect(CGRect, radius: CGFloat)`](https://developer.apple.com/documentation/uikit/uipointershape-swift.enum/roundedrect\(_:radius:\))
The pointer morphs into a rounded rectangle using the provided corner radius.
### [Accessing corner radius](https://developer.apple.com/documentation/uikit/uipointershape-swift.enum#Accessing-corner-radius)
[`static let defaultCornerRadius: CGFloat`](https://developer.apple.com/documentation/uikit/uipointershape-swift.enum/defaultcornerradius)
The default corner radius for a pointer using a rounded rectangle.
