# wasConnectedNotification
A notification the system posts when a new capture device becomes available.
iOS 4.0+iPadOS 4.0+Mac Catalyst 14.0+macOS 10.7+tvOS 17.0+visionOS 2.1+
```
class let wasConnectedNotification: [NSNotification](https://developer.apple.com/documentation/Foundation/NSNotification).[Name](https://developer.apple.com/documentation/Foundation/NSNotification/Name-swift.struct)
```

## [Discussion](https://developer.apple.com/documentation/AVFoundation/AVCaptureDevice/wasConnectedNotification#Discussion)
The notification’s [`object`](https://developer.apple.com/documentation/Foundation/Notification/object) property contains the capture device that connected.
