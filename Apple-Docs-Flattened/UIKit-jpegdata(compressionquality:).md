# jpegData(compressionQuality:)
Returns a data object that contains the image in JPEG format.
iOSiPadOSMac CatalysttvOSvisionOSwatchOS 2.0+
```
func jpegData(compressionQuality: [CGFloat](https://developer.apple.com/documentation/CoreFoundation/CGFloat-swift.struct)) -> [Data](https://developer.apple.com/documentation/Foundation/Data)?
```

## [ Parameters ](https://developer.apple.com/documentation/uikit/uiimage/jpegdata\(compressionquality:\)#parameters) 

`compressionQuality`
    
The quality of the resulting JPEG image, expressed as a value from `0.0` to `1.0`. The value `0.0` represents the maximum compression (or lowest quality) while the value `1.0` represents the least compression (or best quality).
## [Return Value](https://developer.apple.com/documentation/uikit/uiimage/jpegdata\(compressionquality:\)#return-value)
A data object containing the JPEG data, or `nil` if there’s a problem generating the data. This function may return `nil` if the image has no data or if the underlying `CGImageRef` contains data in an unsupported bitmap format.
## [Discussion](https://developer.apple.com/documentation/uikit/uiimage/jpegdata\(compressionquality:\)#Discussion)
If the image object’s underlying image data has been purged, calling this function forces that data to be reloaded into memory.
