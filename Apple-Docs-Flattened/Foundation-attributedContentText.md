Instance Property
# attributedContentText
An optional string describing the extension item content.
iOS 8.0+iPadOS 8.0+Mac Catalyst 13.1+macOS 10.10+tvOS 9.0+visionOS 1.0+watchOS 2.0+
```
@NSCopying
var attributedContentText: [NSAttributedString](https://developer.apple.com/documentation/foundation/nsattributedstring)? { get set }
```

## [Discussion](https://developer.apple.com/documentation/Foundation/NSExtensionItem/attributedContentText#Discussion)
Important
Alternatively, you can set attributed content in the [`userInfo`](https://developer.apple.com/documentation/foundation/nsextensionitem/userinfo) dictionary using the [`NSExtensionItemAttributedContentTextKey`](https://developer.apple.com/documentation/foundation/nsextensionitemattributedcontenttextkey) key. However, setting the [`userInfo`](https://developer.apple.com/documentation/foundation/nsextensionitem/userinfo) dictionary after setting [`attributedContentText`](https://developer.apple.com/documentation/foundation/nsextensionitem/attributedcontenttext) overrides this property.
