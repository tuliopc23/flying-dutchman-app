Type Method
# screenRect(fromView:rect:)
Returns the frame in screen coordinates.
macOS 10.10+
```
static func screenRect(
    fromView parentView: [NSView](https://developer.apple.com/documentation/appkit/nsview),
    rect frame: [NSRect](https://developer.apple.com/documentation/Foundation/NSRect)
) -> [NSRect](https://developer.apple.com/documentation/Foundation/NSRect)
```

## [Discussion](https://developer.apple.com/documentation/AppKit/NSAccessibility-swift.struct/screenRect\(fromView:rect:\)#Discussion)
Given a frame in the specified view’s coordinates, it returns the same frame in the screen’s coordinates.
