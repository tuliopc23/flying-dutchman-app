# CLLocationCoordinate2D
The latitude and longitude associated with a location, specified using the WGS 84 reference frame.
iOS 2.0+iPadOS 2.0+Mac Catalyst 13.0+macOS 10.6+tvOS 9.0+visionOS 1.0+watchOS 2.0+
