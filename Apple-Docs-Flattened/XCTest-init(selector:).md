# init(selector:)
Initializes a test case with a selector.
```
init(selector: [Selector](https://developer.apple.com/documentation/ObjectiveC/Selector))
```

## [ Parameters ](https://developer.apple.com/documentation/xctest/xctestcase/init\(selector:\)#parameters) 

`selector`
    
A selector to use when running this test.
