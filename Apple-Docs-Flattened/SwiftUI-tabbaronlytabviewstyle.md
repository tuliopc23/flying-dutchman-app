# TabBarOnlyTabViewStyle
A tab view style that displays a tab bar when possible.
iOS 18.0+iPadOS 18.0+Mac Catalyst 18.0+macOS 15.0+tvOS 18.0+visionOS 2.0+
```
struct TabBarOnlyTabViewStyle
```

## [Overview](https://developer.apple.com/documentation/swiftui/tabbaronlytabviewstyle#overview)
Use [`tabBarOnly`](https://developer.apple.com/documentation/swiftui/tabviewstyle/tabbaronly) to construct this style.
To apply this style to a tab view, or to a view that contains tab views, use the [`tabViewStyle(_:)`](https://developer.apple.com/documentation/swiftui/view/tabviewstyle\(_:\)) modifier.
