# outlineView(_:persistentObjectForItem:)
Invoked by `outlineView` to return an archived object for `item`.
macOS
```
@[MainActor](https://developer.apple.com/documentation/Swift/MainActor)
optional func outlineView(
    _ outlineView: [NSOutlineView](https://developer.apple.com/documentation/appkit/nsoutlineview),
    persistentObjectForItem item: Any?
) -> Any?
```

## [ Parameters ](https://developer.apple.com/documentation/appkit/nsoutlineviewdatasource/outlineview\(_:persistentobjectforitem:\)#parameters) 

`outlineView`
    
The outline view that sent the message. 

`item`
    
The item for which to return an archived object.
## [Return Value](https://developer.apple.com/documentation/appkit/nsoutlineviewdatasource/outlineview\(_:persistentobjectforitem:\)#return-value)
An archived representation of `item`. If the item is an archived object, this method may return the item.
## [Discussion](https://developer.apple.com/documentation/appkit/nsoutlineviewdatasource/outlineview\(_:persistentobjectforitem:\)#Discussion)
When the outline view is saving the expanded items, this method is called for each expanded item, to translate the outline view item to an archived object.
### [Special Considerations](https://developer.apple.com/documentation/appkit/nsoutlineviewdatasource/outlineview\(_:persistentobjectforitem:\)#Special-Considerations)
You must implement this method if you are automatically saving expanded items (that is, if [`autosaveExpandedItems`](https://developer.apple.com/documentation/appkit/nsoutlineview/autosaveexpandeditems) returns [`true`](https://developer.apple.com/documentation/Swift/true)).
