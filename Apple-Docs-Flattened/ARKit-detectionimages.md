Instance Property
# detectionImages
A set of images that ARKit searches for in the user’s environment.
iOS 11.3+iPadOS 11.3+Mac Catalyst 13.1+
```
var detectionImages: [Set](https://developer.apple.com/documentation/Swift/Set)<[ARReferenceImage](https://developer.apple.com/documentation/arkit/arreferenceimage)>! { get set }
```

## [Discussion](https://developer.apple.com/documentation/arkit/arworldtrackingconfiguration/detectionimages#Discussion)
Add members to this set for each image that ARKit searches for in the user’s environment. When ARKit _detects_ a matching image, the framework creates an [`ARImageAnchor`](https://developer.apple.com/documentation/arkit/arimageanchor) object and adds it to the session.
To define the reference images that this property contains, create an asset catalog in Xcode or create [`ARReferenceImage`](https://developer.apple.com/documentation/arkit/arreferenceimage) objects programmatically. For more information, see [Detecting Images in an AR Experience](https://developer.apple.com/documentation/arkit/detecting-images-in-an-ar-experience).
If you set a nonzero value for [`maximumNumberOfTrackedImages`](https://developer.apple.com/documentation/arkit/arworldtrackingconfiguration/maximumnumberoftrackedimages), ARKit enables image _tracking_ , which continuously updates the transform for up to four of the reference image anchors as the session progresses. For an example, see [Tracking and altering images](https://developer.apple.com/documentation/arkit/tracking-and-altering-images).
### [Limit Reference Images for Performance](https://developer.apple.com/documentation/arkit/arworldtrackingconfiguration/detectionimages#Limit-Reference-Images-for-Performance)
Image detection accuracy and performance may decline as the number of images in this set increases. For best results, limit your detection image count to no more than around 100.
To detect more than 100 images, your app can allocate a certain amount of time for the first 100 images before moving on to the next 100, and so on. When you update the contents of this property, call [`run(_:options:)`](https://developer.apple.com/documentation/arkit/arsession/run\(_:options:\)) again with your app’s configuration to effect the change.
