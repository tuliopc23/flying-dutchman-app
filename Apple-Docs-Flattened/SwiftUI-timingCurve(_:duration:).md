# timingCurve(_:duration:)
Creates a new animation with speed controlled by the given curve.
iOS 17.0+iPadOS 17.0+Mac Catalyst 17.0+macOS 14.0+tvOS 17.0+visionOS 1.0+watchOS 10.0+
```
static func timingCurve(
    _ curve: [UnitCurve](https://developer.apple.com/documentation/swiftui/unitcurve),
    duration: [TimeInterval](https://developer.apple.com/documentation/Foundation/TimeInterval)
) -> [Animation](https://developer.apple.com/documentation/swiftui/animation)
```

## [ Parameters ](https://developer.apple.com/documentation/SwiftUI/Animation/timingCurve\(_:duration:\)#parameters)
