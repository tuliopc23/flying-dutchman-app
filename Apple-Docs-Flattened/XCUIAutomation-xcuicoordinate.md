# XCUICoordinate
A location on screen relative to a UI element.
iOSiPadOSMac CatalystmacOSvisionOSwatchOSXcode 16.3+
```
@[MainActor](https://developer.apple.com/documentation/Swift/MainActor)
class XCUICoordinate
```

## [Overview](https://developer.apple.com/documentation/xcuiautomation/xcuicoordinate#overview)
Coordinates are dynamic, like the elements to which they refer, and may compute different screen locations at different times, or be invalid if the element they reference doesn’t exist.
## [Topics](https://developer.apple.com/documentation/xcuiautomation/xcuicoordinate#topics)
### [Getting coordinate properties](https://developer.apple.com/documentation/xcuiautomation/xcuicoordinate#Getting-coordinate-properties)
[`var referencedElement: XCUIElement`](https://developer.apple.com/documentation/xcuiautomation/xcuicoordinate/referencedelement)
The element that the coordinate is based on, either directly or through the coordinate from which it was derived.
[`var screenPoint: CGPoint`](https://developer.apple.com/documentation/xcuiautomation/xcuicoordinate/screenpoint)
The dynamically computed value of the coordinate’s location on screen.
