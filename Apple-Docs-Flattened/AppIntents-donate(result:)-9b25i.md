# donate(result:)
Donates the intent and optional result to the transcript.
iOS 16.0+iPadOS 16.0+Mac CatalystmacOS 13.0+tvOS 16.0+visionOSwatchOS 9.0+
```
@discardableResult
func donate(result: some [IntentResult](https://developer.apple.com/documentation/appintents/intentresult)) -> [IntentDonationIdentifier](https://developer.apple.com/documentation/appintents/intentdonationidentifier)
```

## [Discussion](https://developer.apple.com/documentation/appintents/appintent/donate\(result:\)-9b25i#discussion)
This synchronous method is available to applications that haven’t adopted Swift async concurrency. The system ignores any exceptions encountered when donating this intent.
