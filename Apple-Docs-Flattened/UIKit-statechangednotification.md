# stateChangedNotification
A notification the document object posts when there’s a change in the state of the document.
iOS 5.0+iPadOS 5.0+Mac Catalyst 13.1+visionOS 1.0+
```
nonisolated
class let stateChangedNotification: [NSNotification](https://developer.apple.com/documentation/Foundation/NSNotification).[Name](https://developer.apple.com/documentation/Foundation/NSNotification/Name-swift.struct)
```

## [Discussion](https://developer.apple.com/documentation/uikit/uidocument/statechangednotification#Discussion)
When handling this notification, check the value of the [`documentState`](https://developer.apple.com/documentation/uikit/uidocument/documentstate) property to see what the new state is, and then proceed accordingly. There’s no `userInfo` dictionary.
