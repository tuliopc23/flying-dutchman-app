# TouchBar
A container for a view that you can show in the Touch Bar.
macOS 10.15+
```
struct TouchBar<Content> where Content : [View](https://developer.apple.com/documentation/swiftui/view)
```

Creates a non-customizable Touch Bar view container.
[`init(id: String, content: () -> Content)`](https://developer.apple.com/documentation/swiftui/touchbar/init\(id:content:\))
Creates a customizable Touch Bar view container with a globally unique identifier.
