# dynamicTypeSize
The current Dynamic Type size.
iOS 15.0+iPadOS 15.0+Mac Catalyst 15.0+macOS 12.0+tvOS 15.0+visionOS 1.0+watchOS 8.0+
```
var dynamicTypeSize: [DynamicTypeSize](https://developer.apple.com/documentation/swiftui/dynamictypesize) { get set }
```

## [Discussion](https://developer.apple.com/documentation/swiftui/environmentvalues/dynamictypesize#discussion)
This value changes as the user’s chosen Dynamic Type size changes. The default value is device-dependent.
When limiting the Dynamic Type size, consider if adding a large content view with [`accessibilityShowsLargeContentViewer()`](https://developer.apple.com/documentation/swiftui/view/accessibilityshowslargecontentviewer\(\)) would be appropriate.
On macOS, this value cannot be changed by users and does not affect the text size.
