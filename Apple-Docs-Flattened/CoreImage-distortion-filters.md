# Distortion Filters
Apply distortion to images.
## [Topics](https://developer.apple.com/documentation/coreimage/distortion-filters#topics)
### [Filters](https://developer.apple.com/documentation/coreimage/distortion-filters#Filters)
[`class func bumpDistortion() -> any CIFilter & CIBumpDistortion`](https://developer.apple.com/documentation/coreimage/cifilter-swift.class/bumpdistortion\(\))
Distorts an image with a concave or convex bump.
[`class func bumpDistortionLinear() -> any CIFilter & CIBumpDistortionLinear`](https://developer.apple.com/documentation/coreimage/cifilter-swift.class/bumpdistortionlinear\(\))
Linearly distorts an image with a concave or convex bump.
[`class func circleSplashDistortion() -> any CIFilter & CICircleSplashDistortion`](https://developer.apple.com/documentation/coreimage/cifilter-swift.class/circlesplashdistortion\(\))
Distorts an image with radiating circles to the periphery of the image.
[`class func circularWrap() -> any CIFilter & CICircularWrap`](https://developer.apple.com/documentation/coreimage/cifilter-swift.class/circularwrap\(\))
Distorts an image by increasing the distance of the center of the image.
[`class func displacementDistortion() -> any CIFilter & CIDisplacementDistortion`](https://developer.apple.com/documentation/coreimage/cifilter-swift.class/displacementdistortion\(\))
Applies the grayscale values of the second image to the first image.
[`class func droste() -> any CIFilter & CIDroste`](https://developer.apple.com/documentation/coreimage/cifilter-swift.class/droste\(\))
Stylizes an image with the Droste effect.
[`class func glassDistortion() -> any CIFilter & CIGlassDistortion`](https://developer.apple.com/documentation/coreimage/cifilter-swift.class/glassdistortion\(\))
Distorts an image by applying a glass-like texture.
[`class func glassLozenge() -> any CIFilter & CIGlassLozenge`](https://developer.apple.com/documentation/coreimage/cifilter-swift.class/glasslozenge\(\))
Creates a lozenge-shaped lens and distorts the image.
[`class func holeDistortion() -> any CIFilter & CIHoleDistortion`](https://developer.apple.com/documentation/coreimage/cifilter-swift.class/holedistortion\(\))
Distorts an image with a circular area that pushes the image outward.
[`class func lightTunnel() -> any CIFilter & CILightTunnel`](https://developer.apple.com/documentation/coreimage/cifilter-swift.class/lighttunnel\(\))
Distorts an image by generating a light tunnel.
[`class func ninePartStretched() -> any CIFilter & CINinePartStretched`](https://developer.apple.com/documentation/coreimage/cifilter-swift.class/ninepartstretched\(\))
Distorts an image by stretching it between two breakpoints.
[`class func ninePartTiled() -> any CIFilter & CINinePartTiled`](https://developer.apple.com/documentation/coreimage/cifilter-swift.class/nineparttiled\(\))
Distorts an image by tiling portions of it.
[`class func pinchDistortion() -> any CIFilter & CIPinchDistortion`](https://developer.apple.com/documentation/coreimage/cifilter-swift.class/pinchdistortion\(\))
Distorts an image by creating a pinch effect with stronger distortion in the center.
[`class func stretchCrop() -> any CIFilter & CIStretchCrop`](https://developer.apple.com/documentation/coreimage/cifilter-swift.class/stretchcrop\(\))
Distorts an image by stretching or cropping to fit a specified size.
[`class func torusLensDistortion() -> any CIFilter & CITorusLensDistortion`](https://developer.apple.com/documentation/coreimage/cifilter-swift.class/toruslensdistortion\(\))
Creates a torus-shaped lens to distort the image.
[`class func twirlDistortion() -> any CIFilter & CITwirlDistortion`](https://developer.apple.com/documentation/coreimage/cifilter-swift.class/twirldistortion\(\))
Distorts an image by rotating pixels around a center point.
[`class func vortexDistortion() -> any CIFilter & CIVortexDistortion`](https://developer.apple.com/documentation/coreimage/cifilter-swift.class/vortexdistortion\(\))
Distorts an image by using a vortex effect created by rotating pixels around a point.
