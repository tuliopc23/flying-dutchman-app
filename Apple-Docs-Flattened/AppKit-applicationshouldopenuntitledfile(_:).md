# applicationShouldOpenUntitledFile(_:)
Returns a Boolean value that indicates if the app can open an untitled file.
macOS
```
@[MainActor](https://developer.apple.com/documentation/Swift/MainActor)
optional func applicationShouldOpenUntitledFile(_ sender: [NSApplication](https://developer.apple.com/documentation/appkit/nsapplication)) -> [Bool](https://developer.apple.com/documentation/Swift/Bool)
```

## [ Parameters ](https://developer.apple.com/documentation/appkit/nsapplicationdelegate/applicationshouldopenuntitledfile\(_:\)#parameters) 

`sender`
    
The application object associated with the delegate.
## [Return Value](https://developer.apple.com/documentation/appkit/nsapplicationdelegate/applicationshouldopenuntitledfile\(_:\)#return-value)
[`true`](https://developer.apple.com/documentation/Swift/true) if the application should open a new untitled file or [`false`](https://developer.apple.com/documentation/Swift/false) if it should not.
## [Discussion](https://developer.apple.com/documentation/appkit/nsapplicationdelegate/applicationshouldopenuntitledfile\(_:\)#Discussion)
Use this method to decide whether the application should open a new, untitled file. Note that [`applicationOpenUntitledFile(_:)`](https://developer.apple.com/documentation/appkit/nsapplicationdelegate/applicationopenuntitledfile\(_:\)) is invoked if this method returns [`true`](https://developer.apple.com/documentation/Swift/true).
