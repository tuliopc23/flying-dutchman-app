# init(cvPixelBuffer:)
Initializes an image object from the contents of a Core Video pixel buffer.
iOS 5.0+iPadOS 5.0+Mac Catalyst 13.1+macOS 10.11+tvOSvisionOS 1.0+
```
init(cvPixelBuffer pixelBuffer: [CVPixelBuffer](https://developer.apple.com/documentation/CoreVideo/CVPixelBuffer))
```

## [ Parameters ](https://developer.apple.com/documentation/CoreImage/CIImage/init\(cvPixelBuffer:\)#parameters) 

`pixelBuffer`
    
A `CVPixelBuffer` object.
## [Return Value](https://developer.apple.com/documentation/CoreImage/CIImage/init\(cvPixelBuffer:\)#return-value)
The initialized image object.
