# naturalTimeScale
The natural time scale of the media that a track references.
iOS 15.0+iPadOS 15.0+Mac CatalystmacOS 12.0+tvOS 15.0+visionOS 1.0+watchOS 8.0+
```
static var naturalTimeScale: [AVAsyncProperty](https://developer.apple.com/documentation/avfoundation/avasyncproperty)<Root, [CMTimeScale](https://developer.apple.com/documentation/CoreMedia/CMTimeScale)> { get }
```

Available when `Root` inherits `AVAssetTrack`.
## [Discussion](https://developer.apple.com/documentation/AVFoundation/AVPartialAsyncProperty/naturalTimeScale#Discussion)
Use the [`load(_:isolation:)`](https://developer.apple.com/documentation/avfoundation/avasynchronouskeyvalueloading/load\(_:isolation:\)) method to retrieve the property value.
