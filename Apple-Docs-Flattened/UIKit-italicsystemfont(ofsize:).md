# italicSystemFont(ofSize:)
Returns the font object for standard interface items in italic type in the specified size.
iOS 2.0+iPadOS 2.0+Mac Catalyst 13.1+tvOSvisionOS 1.0+watchOS 2.0+
```
class func italicSystemFont(ofSize fontSize: [CGFloat](https://developer.apple.com/documentation/CoreFoundation/CGFloat-swift.struct)) -> [UIFont](https://developer.apple.com/documentation/uikit/uifont)
```

## [ Parameters ](https://developer.apple.com/documentation/uikit/uifont/italicsystemfont\(ofsize:\)#parameters) 

`fontSize`
    
The size (in points) for the font. This value must be greater than `0.0`.
## [Return Value](https://developer.apple.com/documentation/uikit/uifont/italicsystemfont\(ofsize:\)#return-value)
A font object of the specified size.
## [Discussion](https://developer.apple.com/documentation/uikit/uifont/italicsystemfont\(ofsize:\)#Discussion)
Instead of using this method to get a font, it’s often more appropriate to use [`preferredFont(forTextStyle:)`](https://developer.apple.com/documentation/uikit/uifont/preferredfont\(fortextstyle:\)) because that method respects the user’s selected content size category.
