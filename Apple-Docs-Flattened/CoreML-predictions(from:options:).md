# predictions(from:options:)
Generates a prediction for each input feature provider within the batch provider using the prediction options.
iOS 12.0+iPadOS 12.0+Mac Catalyst 13.1+macOS 10.14+tvOS 12.0+visionOS 1.0+watchOS 5.0+
```
func predictions(
    from inputBatch: any [MLBatchProvider](https://developer.apple.com/documentation/coreml/mlbatchprovider),
    options: [MLPredictionOptions](https://developer.apple.com/documentation/coreml/mlpredictionoptions)
) throws -> any [MLBatchProvider](https://developer.apple.com/documentation/coreml/mlbatchprovider)
```

## [ Parameters ](https://developer.apple.com/documentation/CoreML/MLModel/predictions\(from:options:\)#parameters) 

`inputBatch`
    
A batch provider that contains multiple input feature providers. The model makes a prediction for each feature provider. 

`options`
    
The runtime settings the model uses as it makes a prediction.
## [Return Value](https://developer.apple.com/documentation/CoreML/MLModel/predictions\(from:options:\)#return-value)
A batch provider that contains an output feature provider for each prediction.
## [Discussion](https://developer.apple.com/documentation/CoreML/MLModel/predictions\(from:options:\)#Discussion)
Use this method to make more than one prediction at one time.
