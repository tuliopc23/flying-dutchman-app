# placeholderRect(forBounds:)
Returns the drawing rectangle for the text field’s placeholder text.
iOS 2.0+iPadOS 2.0+Mac Catalyst 13.1+tvOSvisionOS 1.0+
```
@[MainActor](https://developer.apple.com/documentation/Swift/MainActor)
func placeholderRect(forBounds bounds: [CGRect](https://developer.apple.com/documentation/CoreFoundation/CGRect)) -> [CGRect](https://developer.apple.com/documentation/CoreFoundation/CGRect)
```

## [ Parameters ](https://developer.apple.com/documentation/uikit/uitextfield/placeholderrect\(forbounds:\)#parameters) 

`bounds`
    
The bounding rectangle of the receiver.
## [Return Value](https://developer.apple.com/documentation/uikit/uitextfield/placeholderrect\(forbounds:\)#return-value)
The computed drawing rectangle for the placeholder text.
## [Discussion](https://developer.apple.com/documentation/uikit/uitextfield/placeholderrect\(forbounds:\)#Discussion)
You should not call this method directly. If you want to customize the drawing rectangle for the placeholder text, you can override this method and return a different rectangle.
If the placeholder string is empty or `nil`, this method is not called.
