## [ Parameters ](https://developer.apple.com/documentation/foundationmodels/languagemodelsession/respond\(to:options:\)-b2re#parameters) 

`prompt`
    
A prompt for the model to respond to. 

`options`
    
GenerationOptions that control how tokens are sampled from the distribution the model produces.
## [Return Value](https://developer.apple.com/documentation/foundationmodels/languagemodelsession/respond\(to:options:\)-b2re#return-value)
