# requestConfirmation(conditions:actionName:dialog:)
Requests user confirmation before performing the app intent.
iOS 18.0+iPadOS 18.0+Mac CatalystmacOS 15.0+tvOS 18.0+visionOS 2.0+watchOS 11.0+
```
func requestConfirmation(
    conditions: [ConfirmationConditions](https://developer.apple.com/documentation/appintents/confirmationconditions) = [],
    actionName: [ConfirmationActionName](https://developer.apple.com/documentation/appintents/confirmationactionname) = .`continue`,
    dialog: [IntentDialog](https://developer.apple.com/documentation/appintents/intentdialog)
) async throws
```

## [ Parameters ](https://developer.apple.com/documentation/AppIntents/AppIntent/requestConfirmation\(conditions:actionName:dialog:\)#parameters) 

`conditions`
    
The preconditions for requesting confirmation. 

`actionName`
    
The name for the confirmation action. 

`dialog`
    
The confirmation dialog.
