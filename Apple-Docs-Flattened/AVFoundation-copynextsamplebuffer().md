# copyNextSampleBuffer()
Copies the next sample buffer from the output.
iOS 4.1+iPadOS 4.1+Mac Catalyst 13.1+macOS 10.7–11.0DeprecatedtvOS 9.0–26.2DeprecatedvisionOS 1.0+
```
func copyNextSampleBuffer() -> [CMSampleBuffer](https://developer.apple.com/documentation/CoreMedia/CMSampleBuffer)?
```

## [Return Value](https://developer.apple.com/documentation/avfoundation/avassetreaderoutput/copynextsamplebuffer\(\)#return-value)
The output sample buffer, or `nil` if you’ve read all samples or an error occurs.
## [Discussion](https://developer.apple.com/documentation/avfoundation/avassetreaderoutput/copynextsamplebuffer\(\)#Discussion)
This method returns `nil` when you’ve read all available sample buffers, or if there’s an error. Check the value of the asset reader’s [`status`](https://developer.apple.com/documentation/avfoundation/avassetreader/status-swift.property) property to determine the reason.
