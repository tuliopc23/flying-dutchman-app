# Color Effect Filters
Apply color effects, including photo effects, dithering, and color maps.
## [Topics](https://developer.apple.com/documentation/coreimage/color-effect-filters#topics)
### [Color Effect Filters](https://developer.apple.com/documentation/coreimage/color-effect-filters#Color-Effect-Filters)

[`class func colorCurves() -> any CIFilter & CIColorCurves`](https://developer.apple.com/documentation/coreimage/cifilter-swift.class/colorcurves\(\))
Adjusts an image’s color curves.

Adjusts an image’s colors to shades of a single color.
[`class func colorPosterize() -> any CIFilter & CIColorPosterize`](https://developer.apple.com/documentation/coreimage/cifilter-swift.class/colorposterize\(\))
Flattens an image’s colors.

[`class func photoEffectInstant() -> any CIFilter & CIPhotoEffect`](https://developer.apple.com/documentation/coreimage/cifilter-swift.class/photoeffectinstant\(\))

[`class func photoEffectMono() -> any CIFilter & CIPhotoEffect`](https://developer.apple.com/documentation/coreimage/cifilter-swift.class/photoeffectmono\(\))

[`class func photoEffectNoir() -> any CIFilter & CIPhotoEffect`](https://developer.apple.com/documentation/coreimage/cifilter-swift.class/photoeffectnoir\(\))

[`class func photoEffectProcess() -> any CIFilter & CIPhotoEffect`](https://developer.apple.com/documentation/coreimage/cifilter-swift.class/photoeffectprocess\(\))
Lowers the contrast of the input image.
[`class func photoEffectTonal() -> any CIFilter & CIPhotoEffect`](https://developer.apple.com/documentation/coreimage/cifilter-swift.class/photoeffecttonal\(\))
Adjusts an image’s colors to black and white.
[`class func photoEffectTransfer() -> any CIFilter & CIPhotoEffect`](https://developer.apple.com/documentation/coreimage/cifilter-swift.class/photoeffecttransfer\(\))
Brightens an image’s colors.
[`class func sepiaTone() -> any CIFilter & CISepiaTone`](https://developer.apple.com/documentation/coreimage/cifilter-swift.class/sepiatone\(\))
Adjusts an image’s colors to shades of brown.
[`class func thermal() -> any CIFilter & CIThermal`](https://developer.apple.com/documentation/coreimage/cifilter-swift.class/thermal\(\))
Alters the image to make it look like it was taken by a thermal camera.
[`class func vignette() -> any CIFilter & CIVignette`](https://developer.apple.com/documentation/coreimage/cifilter-swift.class/vignette\(\))

[`class func vignetteEffect() -> any CIFilter & CIVignetteEffect`](https://developer.apple.com/documentation/coreimage/cifilter-swift.class/vignetteeffect\(\))
