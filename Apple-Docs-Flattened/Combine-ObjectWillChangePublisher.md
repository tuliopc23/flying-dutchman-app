# ObjectWillChangePublisher
The type of publisher that emits before the object has changed.
iOS 13.0+iPadOS 13.0+Mac Catalyst 13.0+macOS 10.15+tvOS 13.0+visionOS 1.0+watchOS 6.0+
