# init(uniformTrailingCorners:topLeadingCorner:bottomLeadingCorner:)
Create a rectangle with a corner style set on the trailing two corners uniformly, and two other styles for the leading two corners respectively.
iOS 26.0+iPadOS 26.0+Mac Catalyst 26.0+macOS 26.0+tvOS 26.0+visionOS 26.0+watchOS 26.0+
```
init(
    uniformTrailingCorners: [Edge](https://developer.apple.com/documentation/swiftui/edge).[Corner](https://developer.apple.com/documentation/swiftui/edge/corner).[Style](https://developer.apple.com/documentation/swiftui/edge/corner/style) = .concentric,
    topLeadingCorner: [Edge](https://developer.apple.com/documentation/swiftui/edge).[Corner](https://developer.apple.com/documentation/swiftui/edge/corner).[Style](https://developer.apple.com/documentation/swiftui/edge/corner/style) = .concentric,
    bottomLeadingCorner: [Edge](https://developer.apple.com/documentation/swiftui/edge).[Corner](https://developer.apple.com/documentation/swiftui/edge/corner).[Style](https://developer.apple.com/documentation/swiftui/edge/corner/style) = .concentric
)
```

## [ Parameters ](https://developer.apple.com/documentation/swiftui/concentricrectangle/init\(uniformtrailingcorners:topleadingcorner:bottomleadingcorner:\)#parameters) 

`uniformTrailingCorners`
    
The corner style to be applied to the trailing two corners uniformly. 

`topLeadingCorner`
    
The corner style for the top leading corner. 

`bottomLeadingCorner`
    
The corner style for the bottom leading corner.
