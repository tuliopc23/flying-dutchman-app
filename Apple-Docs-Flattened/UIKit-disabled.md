# disabled
The disabled state of a control.
iOSiPadOSMac CatalysttvOSvisionOS
```
static var disabled: [UIControl](https://developer.apple.com/documentation/uikit/uicontrol).[State](https://developer.apple.com/documentation/uikit/uicontrol/state-swift.struct) { get }
```

## [Discussion](https://developer.apple.com/documentation/uikit/uicontrol/state-swift.struct/disabled#Discussion)
User interactions with disabled control have no effect and the control draws itself with a dimmed appearance to reflect that it’s disabled. You can retrieve and set this value through the [`isEnabled`](https://developer.apple.com/documentation/uikit/uicontrol/isenabled) property.
