# transitionController(forDocumentURL:)
Creates a transition controller that provides the standard system-loading and segue animations for the document browser.
iOS 11.0–12.0DeprecatediPadOS 11.0–12.0DeprecatedMac Catalyst 13.1–13.1Deprecated
```
@[MainActor](https://developer.apple.com/documentation/Swift/MainActor)
func transitionController(forDocumentURL documentURL: [URL](https://developer.apple.com/documentation/Foundation/URL)) -> [UIDocumentBrowserTransitionController](https://developer.apple.com/documentation/uikit/uidocumentbrowsertransitioncontroller)
```

Deprecated
Use [`transitionController(forDocumentAt:)`](https://developer.apple.com/documentation/uikit/uidocumentbrowserviewcontroller/transitioncontroller\(fordocumentat:\)) instead.
## [ Parameters ](https://developer.apple.com/documentation/uikit/uidocumentbrowserviewcontroller/transitioncontroller\(fordocumenturl:\)#parameters) 

`documentURL`
    
The URL of a document. Only use URLs provided by the document browser (for example, URLs passed to the delegate’s [`documentBrowser(_:didRequestDocumentCreationWithHandler:)`](https://developer.apple.com/documentation/uikit/uidocumentbrowserviewcontrollerdelegate/documentbrowser\(_:didrequestdocumentcreationwithhandler:\))method’s completion block).
## [Return Value](https://developer.apple.com/documentation/uikit/uidocumentbrowserviewcontroller/transitioncontroller\(fordocumenturl:\)#return-value)
Returns a newly instantiated transition controller. Its [`loadingProgress`](https://developer.apple.com/documentation/uikit/uidocumentbrowsertransitioncontroller/loadingprogress) and [`targetView`](https://developer.apple.com/documentation/uikit/uidocumentbrowsertransitioncontroller/targetview) properties are both set to `nil`.
## [Discussion](https://developer.apple.com/documentation/uikit/uidocumentbrowserviewcontroller/transitioncontroller\(fordocumenturl:\)#Discussion)
For the animations to function properly, you must maintain a strong reference to the transition controller until all the animation sequences are complete.
For more about using the transition controller, see [`UIDocumentBrowserTransitionController`](https://developer.apple.com/documentation/uikit/uidocumentbrowsertransitioncontroller).
