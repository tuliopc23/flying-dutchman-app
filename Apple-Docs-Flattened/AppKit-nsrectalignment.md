Enumeration
# NSRectAlignment
Constants that specify alignment to an edge or a set of edges depending on the user interface layout direction.
macOS 10.15+
```
enum NSRectAlignment
```

[`case topLeading`](https://developer.apple.com/documentation/appkit/nsrectalignment/topleading)

[`case leading`](https://developer.apple.com/documentation/appkit/nsrectalignment/leading)

[`case bottomTrailing`](https://developer.apple.com/documentation/appkit/nsrectalignment/bottomtrailing)
Aligns to the bottom and trailing edges.
[`case trailing`](https://developer.apple.com/documentation/appkit/nsrectalignment/trailing)
Aligns to the trailing edge.
[`case topTrailing`](https://developer.apple.com/documentation/appkit/nsrectalignment/toptrailing)
