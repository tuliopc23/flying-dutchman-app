# init(value:in:onEditingChanged:)
Creates a slider to select a value from a given range.
iOS 13.0+iPadOS 13.0+Mac Catalyst 13.0+macOS 10.15+visionOS 1.0+watchOS 6.0+
```
nonisolated
init<V>(
    value: [Binding](https://developer.apple.com/documentation/swiftui/binding)<V>,
    in bounds: [ClosedRange](https://developer.apple.com/documentation/Swift/ClosedRange)<V> = 0...1,
    onEditingChanged: @escaping ([Bool](https://developer.apple.com/documentation/Swift/Bool)) -> [Void](https://developer.apple.com/documentation/Swift/Void) = { _ in }
) where V : [BinaryFloatingPoint](https://developer.apple.com/documentation/Swift/BinaryFloatingPoint), V.[Stride](https://developer.apple.com/documentation/Swift/Strideable/Stride) : [BinaryFloatingPoint](https://developer.apple.com/documentation/Swift/BinaryFloatingPoint)
```

Available when `Label` is `EmptyView` and `ValueLabel` is `EmptyView`.
## [ Parameters ](https://developer.apple.com/documentation/swiftui/slider/init\(value:in:oneditingchanged:\)#parameters) 

`value`
    
The selected value within `bounds`. 

`bounds`
    
The range of the valid values. Defaults to `0...1`. 

`onEditingChanged`
    
A callback for when editing begins and ends.
## [Discussion](https://developer.apple.com/documentation/swiftui/slider/init\(value:in:oneditingchanged:\)#discussion)
The `value` of the created instance is equal to the position of the given value within `bounds`, mapped into `0...1`.
The slider calls `onEditingChanged` when editing begins and ends. For example, on iOS, editing begins when the user starts to drag the thumb along the slider’s track.
