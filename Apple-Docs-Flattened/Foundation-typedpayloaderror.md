# NSUserActivity.TypedPayloadError
An enumeration that describes the error types for getting and setting a typed payload.
iOS 14.0+iPadOS 14.0+Mac Catalyst 14.0+macOS 11.0+tvOS 14.0+visionOS 1.0+watchOS 7.0+
```
enum TypedPayloadError
```

## [Overview](https://developer.apple.com/documentation/foundation/nsuseractivity/typedpayloaderror#overview)
Use this enumeration to manage errors from [`typedPayload(_:)`](https://developer.apple.com/documentation/foundation/nsuseractivity/typedpayload\(_:\)) and [`setTypedPayload(_:)`](https://developer.apple.com/documentation/foundation/nsuseractivity/settypedpayload\(_:\)).
## [Topics](https://developer.apple.com/documentation/foundation/nsuseractivity/typedpayloaderror#topics)
### [Typed payload errors](https://developer.apple.com/documentation/foundation/nsuseractivity/typedpayloaderror#Typed-payload-errors)
[`case encodingError`](https://developer.apple.com/documentation/foundation/nsuseractivity/typedpayloaderror/encodingerror)
An encoding error that indicates that the content failed to encode into a valid dictionary.
[`case invalidContent`](https://developer.apple.com/documentation/foundation/nsuseractivity/typedpayloaderror/invalidcontent)
A decoding error that indicates that the user info dictionary is empty or invalid.
