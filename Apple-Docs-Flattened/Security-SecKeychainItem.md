# SecKeychainItem
An opaque type that represents a keychain item.
macOS 10.0+
```
class SecKeychainItem
```

## [Overview](https://developer.apple.com/documentation/Security/SecKeychainItem#overview)
A [`SecKeychainItem`](https://developer.apple.com/documentation/security/seckeychainitem) object for a certificate that is stored in a keychain can be safely cast to a [`SecCertificate`](https://developer.apple.com/documentation/security/seccertificate) for use with [Certificate, Key, and Trust Services](https://developer.apple.com/documentation/security/certificate-key-and-trust-services).
