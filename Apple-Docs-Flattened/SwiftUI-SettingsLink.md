# SettingsLink
A view that opens the Settings scene defined by an app.
macOS 14.0+
```
struct SettingsLink<Label> where Label : [View](https://developer.apple.com/documentation/swiftui/view)
```

## [Overview](https://developer.apple.com/documentation/SwiftUI/SettingsLink#overview)
On macOS, clicking on the link opens the window for the scene or orders it to the front if it is already open.
## [Topics](https://developer.apple.com/documentation/SwiftUI/SettingsLink#topics)

Creates a settings link with the default system label.
