# didEnterBackgroundNotification
A notification that posts when the app enters the background.
iOS 4.0+iPadOS 4.0+Mac Catalyst 13.1+tvOSvisionOS 1.0+
```
nonisolated
class let didEnterBackgroundNotification: [NSNotification](https://developer.apple.com/documentation/Foundation/NSNotification).[Name](https://developer.apple.com/documentation/Foundation/NSNotification/Name-swift.struct)
```

## [Discussion](https://developer.apple.com/documentation/uikit/uiapplication/didenterbackgroundnotification#Discussion)
The `object` of the notification is the [`UIApplication`](https://developer.apple.com/documentation/uikit/uiapplication) object. There is no `userInfo` dictionary.
