# FileManager.SearchPathDirectory.itemReplacementDirectory
The constant used to create a temporary directory.
iOS 4.0+iPadOS 4.0+Mac Catalyst 13.1+macOS 10.6+tvOS 9.0+visionOS 1.0+watchOS 2.0+
```
case itemReplacementDirectory
```

## [Discussion](https://developer.apple.com/documentation/Foundation/FileManager/SearchPathDirectory/itemReplacementDirectory#Discussion)
Pass this constant to the [`FileManager`](https://developer.apple.com/documentation/foundation/filemanager) method [`url(for:in:appropriateFor:create:)`](https://developer.apple.com/documentation/foundation/filemanager/url\(for:in:appropriatefor:create:\)) in order to create a temporary directory.
