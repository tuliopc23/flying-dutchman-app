# HoverEffectGroup
Describes a grouping of effects that activate together.
visionOS 2.0+
```
struct HoverEffectGroup
```

Use [`HoverEffectGroup`](https://developer.apple.com/documentation/swiftui/hovereffectgroup) to apply effects to multiple views in concert.
