# setTitle(_:for:)
Sets the title to use for the specified state.
iOS 2.0+iPadOS 2.0+Mac Catalyst 13.1+tvOSvisionOS 1.0+
```
@[MainActor](https://developer.apple.com/documentation/Swift/MainActor)
func setTitle(
    _ title: [String](https://developer.apple.com/documentation/Swift/String)?,
    for state: [UIControl](https://developer.apple.com/documentation/uikit/uicontrol).[State](https://developer.apple.com/documentation/uikit/uicontrol/state-swift.struct)
)
```

## [ Parameters ](https://developer.apple.com/documentation/uikit/uibutton/settitle\(_:for:\)#parameters) 

`title`
    
The title to use for the specified state. 

`state`
    
The state that uses the specified title. [`UIControl.State`](https://developer.apple.com/documentation/uikit/uicontrol/state-swift.struct) describes the possible values.
## [Discussion](https://developer.apple.com/documentation/uikit/uibutton/settitle\(_:for:\)#Discussion)
Use this method to set the title for the button. The title you specify derives its formatting from the button’s associated label object. If you set both a title and an attributed title for the button, the button prefers the use of the attributed title over this one.
At a minimum, set the value for the [`normal`](https://developer.apple.com/documentation/uikit/uicontrol/state-swift.struct/normal) state. If you don’t specify a title for the other states, the button uses the title associated with the [`normal`](https://developer.apple.com/documentation/uikit/uicontrol/state-swift.struct/normal) state. If you don’t set the value for [`normal`](https://developer.apple.com/documentation/uikit/uicontrol/state-swift.struct/normal), then the property defaults to a system value.
Important
When the user interface idiom is [`UIUserInterfaceIdiom.mac`](https://developer.apple.com/documentation/uikit/uiuserinterfaceidiom/mac) and [`behavioralStyle`](https://developer.apple.com/documentation/uikit/uibutton/behavioralstyle) is [`UIBehavioralStyle.mac`](https://developer.apple.com/documentation/uikit/uibehavioralstyle/mac), your app throws an exception if you use this method to set the title for any state other than [`normal`](https://developer.apple.com/documentation/uikit/uicontrol/state-swift.struct/normal).
