# NSWritingToolsResultOptions
Constants to specify what type of content to allow in Writing Tools suggestions or rewrites.
macOS 15.0+
```
struct NSWritingToolsResultOptions
```

## [Overview](https://developer.apple.com/documentation/appkit/nswritingtoolsresultoptions#overview)
When configuring a text view, specify what type of text input you want Writing Tools to deliver to your view. You can ask it to return plain text without any attributes, or you can ask it to apply relevant formatting attributes to the text. You can even encourage it to return items in a list or format them in a table.
## [Topics](https://developer.apple.com/documentation/appkit/nswritingtoolsresultoptions#topics)
### [Getting the output options](https://developer.apple.com/documentation/appkit/nswritingtoolsresultoptions#Getting-the-output-options)
[`static var plainText: NSWritingToolsResultOptions`](https://developer.apple.com/documentation/appkit/nswritingtoolsresultoptions/plaintext)
An option to allow only plain text without any attributes in the returned text.
[`static var richText: NSWritingToolsResultOptions`](https://developer.apple.com/documentation/appkit/nswritingtoolsresultoptions/richtext)
An option to include style attributes consistent with the RTF format in the returned text.
[`static var list: NSWritingToolsResultOptions`](https://developer.apple.com/documentation/appkit/nswritingtoolsresultoptions/list)
An option to allow list-style formatting in the returned text.
[`static var table: NSWritingToolsResultOptions`](https://developer.apple.com/documentation/appkit/nswritingtoolsresultoptions/table)
An option to allow tabular layout attributes in the returned text.
