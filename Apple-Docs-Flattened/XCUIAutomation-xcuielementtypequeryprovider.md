# XCUIElementTypeQueryProvider
A type that provides ready-made queries for locating descendant UI elements.
iOSiPadOSMac CatalystmacOStvOSvisionOSwatchOSXcode 16.3+
```
@[MainActor](https://developer.apple.com/documentation/Swift/MainActor)
protocol XCUIElementTypeQueryProvider
```

## [Overview](https://developer.apple.com/documentation/xcuiautomation/xcuielementtypequeryprovider#overview)
Accessing the properties to get queries for descendant elements on an instance of a conforming class (such as an [`XCUIElement`](https://developer.apple.com/documentation/xcuiautomation/xcuielement) or [`XCUIElementQuery`](https://developer.apple.com/documentation/xcuiautomation/xcuielementquery) instance) is equivalent to calling [`descendants(matching:)`](https://developer.apple.com/documentation/xcuiautomation/xcuielementquery/descendants\(matching:\)) for the corresponding element type.
