Instance Property
# centeredItemIdentifiers
The set of custom items to display in the center of the toolbar.
iOS 16.0+iPadOS 16.0+Mac Catalyst 16.0+macOS 13.0+
```
@[MainActor](https://developer.apple.com/documentation/Swift/MainActor)
var centeredItemIdentifiers: [Set](https://developer.apple.com/documentation/Swift/Set)<[NSToolbarItem](https://developer.apple.com/documentation/appkit/nstoolbaritem).[Identifier](https://developer.apple.com/documentation/appkit/nstoolbaritem/identifier)> { get set }
```

## [Discussion](https://developer.apple.com/documentation/appkit/nstoolbar/centereditemidentifiers#Discussion)
Set this property to the items you want to appear together in the center of the toolbar. Specify the initial order of the items using the [`toolbarDefaultItemIdentifiers(_:)`](https://developer.apple.com/documentation/appkit/nstoolbardelegate/toolbardefaultitemidentifiers\(_:\)) method of your toolbar delegate object.
