Type Property
# number
A style for formatting decimal values.
iOS 15.0+iPadOS 15.0+Mac Catalyst 15.0+macOS 12.0+tvOS 15.0+visionOS 1.0+watchOS 8.0+
```
static var number: [Decimal](https://developer.apple.com/documentation/foundation/decimal).[FormatStyle](https://developer.apple.com/documentation/foundation/decimal/formatstyle) { get }
```

Available when `Self` is `Decimal.FormatStyle`.
## [Discussion](https://developer.apple.com/documentation/Foundation/FormatStyle/number-3luf2#Discussion)
Use this type property when the call point allows the use of [`Decimal.FormatStyle`](https://developer.apple.com/documentation/foundation/decimal/formatstyle). You typically do this when calling the [`formatted(_:)`](https://developer.apple.com/documentation/foundation/decimal/formatted\(_:\)) method of [`Decimal`](https://developer.apple.com/documentation/foundation/decimal).
