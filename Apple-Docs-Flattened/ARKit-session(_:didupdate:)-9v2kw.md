# session(_:didUpdate:)
Provides a newly captured camera image and accompanying AR information to the delegate.
iOS 11.0+iPadOS 11.0+Mac Catalyst 13.1+
```
optional func session(
    _ session: [ARSession](https://developer.apple.com/documentation/arkit/arsession),
    didUpdate frame: [ARFrame](https://developer.apple.com/documentation/arkit/arframe)
)
```

## [ Parameters ](https://developer.apple.com/documentation/arkit/arsessiondelegate/session\(_:didupdate:\)-9v2kw#parameters) 

`session`
    
The session providing information. 

`frame`
    
An object containing the new camera image and AR information.

## [Discussion](https://developer.apple.com/documentation/arkit/arsessiondelegate/session\(_:didupdate:\)-9v2kw#Discussion)
Implement this method if you provide your own display for rendering an AR experience. The provided [`ARFrame`](https://developer.apple.com/documentation/arkit/arframe) object contains the latest image captured from the device camera, which you can render as a scene background, as well as information about camera parameters and anchor transforms you can use for rendering virtual content on top of the camera image.
