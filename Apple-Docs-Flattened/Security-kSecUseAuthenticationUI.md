# kSecUseAuthenticationUI
A key whose value indicates whether the user is prompted for authentication.
iOS 9.0+iPadOS 9.0+Mac Catalyst 13.1+macOS 10.11+tvOS 9.0+visionOS 1.0+watchOS 2.0+
```
let kSecUseAuthenticationUI: [CFString](https://developer.apple.com/documentation/CoreFoundation/CFString)
```

## [Discussion](https://developer.apple.com/documentation/Security/kSecUseAuthenticationUI#Discussion)
The corresponding value is of type [`CFString`](https://developer.apple.com/documentation/CoreFoundation/CFString) and contains one of the values listed in [UI authentication values](https://developer.apple.com/documentation/security/search-attribute-keys-and-values#UI-authentication-values). The value specifies whether or not the user is prompted for authentication, if needed. A default value of [`kSecUseAuthenticationUIAllow`](https://developer.apple.com/documentation/security/ksecuseauthenticationuiallow) is assumed when this key is not present.
Current page is kSecUseAuthenticationUI
