# View configuration
Adjust the characteristics of views in a hierarchy.
## [Overview](https://developer.apple.com/documentation/swiftui/view-configuration#Overview)
SwiftUI enables you to tune the appearance and behavior of views using view modifiers.
![](https://docs-assets.developer.apple.com/published/4f97c3ff1212e8df86b49977696cabfe/view-configuration-hero%402x.png)
Many modifiers apply to specific kinds of views or behaviors, but some apply more generally. For example, you can conditionally hide any view by dynamically setting its opacity, display contextual help when people hover over a view, or request the light or dark appearance for a view.
## [Topics](https://developer.apple.com/documentation/swiftui/view-configuration#topics)
### [Hiding views](https://developer.apple.com/documentation/swiftui/view-configuration#Hiding-views)

[`func hidden() -> some View`](https://developer.apple.com/documentation/swiftui/view/hidden\(\))
