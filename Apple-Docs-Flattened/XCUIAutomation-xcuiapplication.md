# XCUIApplication
A proxy that can launch, monitor, and terminate a test application.
iOSiPadOSMac CatalystmacOStvOSvisionOSwatchOSXcode 16.3+
```
@[MainActor](https://developer.apple.com/documentation/Swift/MainActor)
class XCUIApplication
```

## [ Mentioned in ](https://developer.apple.com/documentation/xcuiautomation/xcuiapplication#mentions)
[](https://developer.apple.com/documentation/xcuiautomation/recording-ui-automation-for-testing)
## [Overview](https://developer.apple.com/documentation/xcuiautomation/xcuiapplication#overview)
Use this class to launch, monitor, and terminate your app in a UI test. Use [`wait(for:timeout:)`](https://developer.apple.com/documentation/xcuiautomation/xcuiapplication/wait\(for:timeout:\)) to launch your app and wait for it to reach an expected state before you check test conditions.
