# SidebarCommands
A built-in set of commands for manipulating window sidebars.
iOS 14.0+iPadOS 14.0+Mac Catalyst 14.0+macOS 11.0+visionOS 1.0+
```
struct SidebarCommands
```

## [Overview](https://developer.apple.com/documentation/swiftui/sidebarcommands#overview)
These commands are optional and can be explicitly requested by passing a value of this type to the [`commands(content:)`](https://developer.apple.com/documentation/swiftui/scene/commands\(content:\)) modifier.
