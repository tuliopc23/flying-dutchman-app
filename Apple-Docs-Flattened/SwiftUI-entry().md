## [Environment Values](https://developer.apple.com/documentation/SwiftUI/Entry\(\)#Environment-Values)
Create [`EnvironmentValues`](https://developer.apple.com/documentation/swiftui/environmentvalues) entries by extending the [`EnvironmentValues`](https://developer.apple.com/documentation/swiftui/environmentvalues) structure with new properties and attaching the @Entry macro to the variable declarations:
```
extension EnvironmentValues {
    @Entry var myCustomValue: String = "Default value"
    @Entry var anotherCustomValue = true
}

```

## [Transaction Values](https://developer.apple.com/documentation/SwiftUI/Entry\(\)#Transaction-Values)
Create [`Transaction`](https://developer.apple.com/documentation/swiftui/transaction) entries by extending the [`Transaction`](https://developer.apple.com/documentation/swiftui/transaction) structure with new properties and attaching the @Entry macro to the variable declarations:
```
extension Transaction {
    @Entry var myCustomValue: String = "Default value"
}

```

## [Container Values](https://developer.apple.com/documentation/SwiftUI/Entry\(\)#Container-Values)

## [Focused Values](https://developer.apple.com/documentation/SwiftUI/Entry\(\)#Focused-Values)
Since the default value for [`FocusedValues`](https://developer.apple.com/documentation/swiftui/focusedvalues) is always `nil`, [`FocusedValues`](https://developer.apple.com/documentation/swiftui/focusedvalues) entries cannot specify a different default value and must have an Optional type.
Create [`FocusedValues`](https://developer.apple.com/documentation/swiftui/focusedvalues) entries by extending the [`FocusedValues`](https://developer.apple.com/documentation/swiftui/focusedvalues) structure with new properties and attaching the @Entry macro to the variable declarations:
```
extension FocusedValues {
    @Entry var myCustomValue: String?
}

```
