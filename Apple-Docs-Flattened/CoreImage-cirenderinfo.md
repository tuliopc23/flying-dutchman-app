# CIRenderInfo
An encapsulation of a render task’s timing, passes, and pixels processed.
iOS 11.0+iPadOS 11.0+Mac Catalyst 13.1+macOS 10.13+tvOS 11.0+visionOS 1.0+
```
class CIRenderInfo
```

## [Overview](https://developer.apple.com/documentation/coreimage/cirenderinfo#overview)
A `CIRenderInfo` object allows Xcode Quick Look to visualize the render graph with detailed timing information.
## [Topics](https://developer.apple.com/documentation/coreimage/cirenderinfo#topics)
### [Instance Properties](https://developer.apple.com/documentation/coreimage/cirenderinfo#Instance-Properties)
[`var kernelExecutionTime: TimeInterval`](https://developer.apple.com/documentation/coreimage/cirenderinfo/kernelexecutiontime)
The amount of time a render spent executing kernels.
[`var passCount: Int`](https://developer.apple.com/documentation/coreimage/cirenderinfo/passcount)
The number of passes the render took.
[`var pixelsProcessed: Int`](https://developer.apple.com/documentation/coreimage/cirenderinfo/pixelsprocessed)
The number of pixels the render produced executing kernels.
