# originFromAnchorTransform
The location and orientation of a hand in world space.
visionOS 1.0+
```
var originFromAnchorTransform: [simd_float4x4](https://developer.apple.com/documentation/simd/simd_float4x4) { get }
```

## [Discussion](https://developer.apple.com/documentation/ARKit/HandAnchor/originFromAnchorTransform#Discussion)
The transform of a hand anchor is positioned at the base of the wrist. ARKit provides transforms of other joints on the hand relative to this root transform.
