# scientific
A notation constant that formats values with scientific notation.
iOS 15.0+iPadOS 15.0+Mac Catalyst 15.0+macOS 12.0+tvOS 15.0+visionOS 1.0+watchOS 8.0+
```
static var scientific: [NumberFormatStyleConfiguration](https://developer.apple.com/documentation/foundation/numberformatstyleconfiguration).[Notation](https://developer.apple.com/documentation/foundation/numberformatstyleconfiguration/notation) { get }
```

## [Discussion](https://developer.apple.com/documentation/Foundation/NumberFormatStyleConfiguration/Notation/scientific#Discussion)
The following example shows the effect of using scientific notation with a format style:
```
let scientific = 12345.formatted(.number
    .notation(.scientific)) // 1.2345E4"



```
