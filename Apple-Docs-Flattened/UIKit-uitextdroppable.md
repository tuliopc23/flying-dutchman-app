# UITextDroppable
The interface that determines if a text view is a drop destination.
iOS 11.0+iPadOS 11.0+Mac Catalyst 13.1+visionOS 1.0+
```
@[MainActor](https://developer.apple.com/documentation/Swift/MainActor)
protocol UITextDroppable : [UITextInput](https://developer.apple.com/documentation/uikit/uitextinput), [UITextPasteConfigurationSupporting](https://developer.apple.com/documentation/uikit/uitextpasteconfigurationsupporting)
```
