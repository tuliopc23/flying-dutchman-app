# AttributeScopes.FoundationAttributes.ReferentConceptAttribute
An attribute that specifies a grammatical agreement concept for substituting pronouns in localized text.
iOS 17.0+iPadOS 17.0+Mac Catalyst 17.0+macOS 14.0+tvOS 17.0+visionOS 1.0+watchOS 10.0+
```
@frozen
enum ReferentConceptAttribute
```

## [Overview](https://developer.apple.com/documentation/foundation/attributescopes/foundationattributes/referentconceptattribute#overview)
Use the [`referentConcept`](https://developer.apple.com/documentation/foundation/attributescopes/foundationattributes/referentconcept) formatting attribute for cases where you need to refer to a person using their preferred pronoun in a string.
For an example of how to use a `referentConcept`, see [`TermOfAddress`](https://developer.apple.com/documentation/foundation/termofaddress).
