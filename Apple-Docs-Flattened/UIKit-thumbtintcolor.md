# thumbTintColor
The color used to tint the default thumb images.
iOS 5.0+iPadOS 5.0+Mac Catalyst 13.1+visionOS 1.0+
```
@[MainActor](https://developer.apple.com/documentation/Swift/MainActor)
var thumbTintColor: [UIColor](https://developer.apple.com/documentation/uikit/uicolor)? { get set }
```

## [Discussion](https://developer.apple.com/documentation/uikit/uislider/thumbtintcolor#Discussion)
Setting this property resets the thumb images back to the slider’s default images; any custom images are released by the slider. Setting this property to `nil` resets the tint color to the default and removes any custom thumb images.
Important
This property isn’t available when the user interface idiom is [`UIUserInterfaceIdiom.mac`](https://developer.apple.com/documentation/uikit/uiuserinterfaceidiom/mac) and [`behavioralStyle`](https://developer.apple.com/documentation/uikit/uislider/behavioralstyle) is [`UIBehavioralStyle.mac`](https://developer.apple.com/documentation/uikit/uibehavioralstyle/mac) — setting it while in this state throws an exception.
