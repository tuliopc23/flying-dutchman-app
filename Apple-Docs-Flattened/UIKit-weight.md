# UIFont.Weight
Constants that represent standard typeface styles.
iOSiPadOSMac CatalysttvOSvisionOSwatchOS 4.0+
```
struct Weight
```

## [Overview](https://developer.apple.com/documentation/uikit/uifont/weight#overview)
Use system-defined constants as interchangeable values for [`weight`](https://developer.apple.com/documentation/uikit/uifontdescriptor/traitkey/weight). Each constant corresponds to a different value that indicates the weight of a font. Use these constants to specify the weight parameter in [`systemFont(ofSize:weight:)`](https://developer.apple.com/documentation/uikit/uifont/systemfont\(ofsize:weight:\)). When providing a weight that doesn’t precisely match a font face in the family, the system locates a face that most closely matches the request.
Note
Font [`familyNames`](https://developer.apple.com/documentation/uikit/uifont/familynames) don’t include all system-defined font constants.
