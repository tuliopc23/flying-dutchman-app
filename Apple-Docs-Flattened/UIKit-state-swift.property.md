# state
The state of the control, specified as a bit mask value.
iOS 2.0+iPadOS 2.0+Mac Catalyst 13.1+tvOSvisionOS 1.0+
```
@[MainActor](https://developer.apple.com/documentation/Swift/MainActor)
var state: [UIControl](https://developer.apple.com/documentation/uikit/uicontrol).[State](https://developer.apple.com/documentation/uikit/uicontrol/state-swift.struct) { get }
```

## [Discussion](https://developer.apple.com/documentation/uikit/uicontrol/state-swift.property#Discussion)
The value of this property is a bitmask of the constants in the [`UIControl.State`](https://developer.apple.com/documentation/uikit/uicontrol/state-swift.struct) type. A control can be in more than one state at a time. For example, it can be focused and highlighted at the same time. You can also get the values for individual states using the properties of this class.
