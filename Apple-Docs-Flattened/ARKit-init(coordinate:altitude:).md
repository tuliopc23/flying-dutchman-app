# init(coordinate:altitude:)
Initializes a location anchor with the given coordinate and altitude.
iOS 14.0+iPadOS 14.0+
```
@nonobjc
convenience init(
    coordinate: [CLLocationCoordinate2D](https://developer.apple.com/documentation/CoreLocation/CLLocationCoordinate2D),
    altitude: [CLLocationDistance](https://developer.apple.com/documentation/CoreLocation/CLLocationDistance)? = nil
)
```

## [ Parameters ](https://developer.apple.com/documentation/arkit/argeoanchor/init\(coordinate:altitude:\)#parameters) 

`coordinate`
    
Lattitude and longitude of the anchor’s geographic location. 

`altitude`
    
Vertical distance, in meters, between this anchor and sea level.
