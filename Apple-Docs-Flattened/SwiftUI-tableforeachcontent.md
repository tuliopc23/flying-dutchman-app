# TableForEachContent
A type of table row content that creates table rows created by iterating over a collection.
iOS 16.0+iPadOS 16.0+Mac Catalyst 16.0+macOS 12.0+visionOS 1.0+
```
struct TableForEachContent<Data> where Data : [RandomAccessCollection](https://developer.apple.com/documentation/Swift/RandomAccessCollection), Data.[Element](https://developer.apple.com/documentation/Swift/Sequence/Element) : [Identifiable](https://developer.apple.com/documentation/Swift/Identifiable)
```

## [Overview](https://developer.apple.com/documentation/swiftui/tableforeachcontent#overview)
You don’t use this type directly. The various `Table.init(_:,...)` initializers create this type as the table’s `Rows` generic type.
To explicitly create dynamic collection-based rows, use [`ForEach`](https://developer.apple.com/documentation/swiftui/foreach) instead.
