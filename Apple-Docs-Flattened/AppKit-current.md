Type Property
# current
Returns the appearance object that’s active on the current thread.
macOS 10.9–12.0Deprecated
```
class var current: [NSAppearance](https://developer.apple.com/documentation/appkit/nsappearance)! { get set }
```

Deprecated
Use [`performAsCurrentDrawingAppearance(_:)`](https://developer.apple.com/documentation/appkit/nsappearance/performascurrentdrawingappearance\(_:\)) to temporarily set the drawing appearance, or [`currentDrawingAppearance`](https://developer.apple.com/documentation/appkit/nsappearance/currentdrawingappearance) to access the current drawing appearance.
## [Return Value](https://developer.apple.com/documentation/AppKit/NSAppearance/current#return-value)
The appearance object that’s set on the current thread.
## [Discussion](https://developer.apple.com/documentation/AppKit/NSAppearance/current#Discussion)
When a UI element draws to the screen, it automatically sets its appearance object to the active appearance on the current thread.
## [See Also](https://developer.apple.com/documentation/AppKit/NSAppearance/current#see-also)
