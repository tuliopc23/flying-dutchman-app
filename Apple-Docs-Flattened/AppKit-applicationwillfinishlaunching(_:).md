# applicationWillFinishLaunching(_:)
Tells the delegate that the app’s initialization is about to complete.
macOS 10.10+
```
@[MainActor](https://developer.apple.com/documentation/Swift/MainActor)
optional func applicationWillFinishLaunching(_ notification: [Notification](https://developer.apple.com/documentation/Foundation/Notification))
```

## [ Parameters ](https://developer.apple.com/documentation/appkit/nsapplicationdelegate/applicationwillfinishlaunching\(_:\)#parameters) 

`notification`
    
A notification named [`willFinishLaunchingNotification`](https://developer.apple.com/documentation/appkit/nsapplication/willfinishlaunchingnotification). Calling the [`object`](https://developer.apple.com/documentation/Foundation/NSNotification/object) method of this notification returns the `NSApplication` object itself.
