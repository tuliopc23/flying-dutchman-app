# UISpringLoadedInteractionContext
The interface an object implements to provide information about a spring-loaded interaction.
iOS 11.0+iPadOS 11.0+Mac Catalyst 13.1+visionOS 1.0+
```
@[MainActor](https://developer.apple.com/documentation/Swift/MainActor)
protocol UISpringLoadedInteractionContext : [NSObjectProtocol](https://developer.apple.com/documentation/ObjectiveC/NSObjectProtocol)
```
