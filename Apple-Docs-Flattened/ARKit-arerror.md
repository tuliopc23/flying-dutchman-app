# ARError
An error reported by ARKit.
iOS 11.0+iPadOS 11.0+Mac Catalyst 13.1+

```
struct ARError
```

### [Identifying an error cause](https://developer.apple.com/documentation/arkit/arerror#Identifying-an-error-cause)
[`enum Code`](https://developer.apple.com/documentation/arkit/arerror/code)
Codes that identify errors in ARKit.
[`static var requestFailed: ARError.Code`](https://developer.apple.com/documentation/arkit/arerror/requestfailed)
An error that indicates a request fails.
[`static var cameraUnauthorized: ARError.Code`](https://developer.apple.com/documentation/arkit/arerror/cameraunauthorized)
An error that indicates the app lacks user permission for the camera.
[`static var fileIOFailed: ARError.Code`](https://developer.apple.com/documentation/arkit/arerror/fileiofailed)
An error that indicates a file access fails to read or write.
[`static var insufficientFeatures: ARError.Code`](https://developer.apple.com/documentation/arkit/arerror/insufficientfeatures)
An error that indicates the framework requires more features to complete a task.
[`static var invalidCollaborationData: ARError.Code`](https://developer.apple.com/documentation/arkit/arerror/invalidcollaborationdata)
An error that indicates the framework fails to parse collaboration data the app receives over the network.
[`static var invalidConfiguration: ARError.Code`](https://developer.apple.com/documentation/arkit/arerror/invalidconfiguration)
An error that indicates the configuration contains ambiguous or erroneous data.
[`static var invalidReferenceImage: ARError.Code`](https://developer.apple.com/documentation/arkit/arerror/invalidreferenceimage)
An error that indicates the framework fails to process a reference image.
[`static var invalidReferenceObject: ARError.Code`](https://developer.apple.com/documentation/arkit/arerror/invalidreferenceobject)
An error that indicates the framework fails to process a reference object.
[`static var invalidWorldMap: ARError.Code`](https://developer.apple.com/documentation/arkit/arerror/invalidworldmap)
An error that indicates the framework fails to process a world map.
[`static var microphoneUnauthorized: ARError.Code`](https://developer.apple.com/documentation/arkit/arerror/microphoneunauthorized)
An error that indicates the app lacks user permission for the microphone.
[`static var objectMergeFailed: ARError.Code`](https://developer.apple.com/documentation/arkit/arerror/objectmergefailed)
An error that indicates the framework fails to merge a detected object.
[`static var sensorFailed: ARError.Code`](https://developer.apple.com/documentation/arkit/arerror/sensorfailed)
An error that indicates a sensor fails to provide required input.
[`static var sensorUnavailable: ARError.Code`](https://developer.apple.com/documentation/arkit/arerror/sensorunavailable)
An error that indicates the framework fails to access a required sensor.
[`static var unsupportedConfiguration: ARError.Code`](https://developer.apple.com/documentation/arkit/arerror/unsupportedconfiguration)
An error that indicates the device lacks support for the session’s configuration.
[`static var worldTrackingFailed: ARError.Code`](https://developer.apple.com/documentation/arkit/arerror/worldtrackingfailed)
An error that indicates when world tracking experiences an unrecoverable problem.
[`static var geoTrackingFailed: ARError.Code`](https://developer.apple.com/documentation/arkit/arerror/geotrackingfailed)
An error that indicates when localization imagery fails to match the device’s camera captures.
[`static var geoTrackingNotAvailableAtLocation: ARError.Code`](https://developer.apple.com/documentation/arkit/arerror/geotrackingnotavailableatlocation)
An error that indicates a location lacks geotracking support.
[`static var locationUnauthorized: ARError.Code`](https://developer.apple.com/documentation/arkit/arerror/locationunauthorized)
An error that indicates the app lacks user permission for the device’s current location.
[`static var highResolutionFrameCaptureFailed: ARError.Code`](https://developer.apple.com/documentation/arkit/arerror/highresolutionframecapturefailed)
An error that indicates a problem in the system’s capture pipeline.
[`static var highResolutionFrameCaptureInProgress: ARError.Code`](https://developer.apple.com/documentation/arkit/arerror/highresolutionframecaptureinprogress)
An error that indicates the system needs to finish a high-resolution frame request before accepting another request.
