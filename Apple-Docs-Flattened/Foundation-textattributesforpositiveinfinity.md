# textAttributesForPositiveInfinity
The text attributes used to display the positive infinity symbol.
iOS 2.0+iPadOS 2.0+Mac Catalyst 13.0+macOS 10.0+tvOS 9.0+visionOS 1.0+watchOS 2.0+
```
var textAttributesForPositiveInfinity: [[String](https://developer.apple.com/documentation/Swift/String) : Any]? { get set }
```

## [Discussion](https://developer.apple.com/documentation/foundation/numberformatter/textattributesforpositiveinfinity#Discussion)
This property is a dictionary that contains the text attributes used to display the positive infinity string.
