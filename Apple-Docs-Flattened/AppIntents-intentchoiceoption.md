# IntentChoiceOption
A structure representing an entry in a list of options for a person to choose from before an app intent resumes its action.
iOS 26.0+iPadOS 26.0+Mac CatalystmacOS 26.0+tvOS 26.0+visionOS 26.0+watchOS 26.0+
```
struct IntentChoiceOption
```

## [Overview](https://developer.apple.com/documentation/appintents/intentchoiceoption#overview)
Each option includes display text and an optional style that influences its visual presentation. For example, an option for deleting a file might use a text and style to communicate its destructive action.
## [Topics](https://developer.apple.com/documentation/appintents/intentchoiceoption#topics)
### [Structures](https://developer.apple.com/documentation/appintents/intentchoiceoption#Structures)
[`struct Style`](https://developer.apple.com/documentation/appintents/intentchoiceoption/style-swift.struct)
Defines the visual style and semantic meaning of an [`IntentChoiceOption`](https://developer.apple.com/documentation/appintents/intentchoiceoption).
### [Initializers](https://developer.apple.com/documentation/appintents/intentchoiceoption#Initializers)
[`init(title: LocalizedStringResource, style: IntentChoiceOption.Style)`](https://developer.apple.com/documentation/appintents/intentchoiceoption/init\(title:style:\))
Creates a new option for a person to choose to continue an app intent.
### [Instance Properties](https://developer.apple.com/documentation/appintents/intentchoiceoption#Instance-Properties)
[`let style: IntentChoiceOption.Style`](https://developer.apple.com/documentation/appintents/intentchoiceoption/style-swift.property)
The style applied to the option, affecting its visual appearance in the system UI.
[`let title: LocalizedStringResource`](https://developer.apple.com/documentation/appintents/intentchoiceoption/title)
The localized text displayed for this option.
### [Type Properties](https://developer.apple.com/documentation/appintents/intentchoiceoption#Type-Properties)
[`static var cancel: IntentChoiceOption`](https://developer.apple.com/documentation/appintents/intentchoiceoption/cancel)
A system-provided option that cancel the app intent.
