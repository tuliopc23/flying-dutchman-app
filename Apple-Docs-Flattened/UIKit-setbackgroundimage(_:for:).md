# setBackgroundImage(_:for:)
Sets the background image to use for the specified button state.
iOS 2.0+iPadOS 2.0+Mac Catalyst 13.1+tvOSvisionOS 1.0+
```
@[MainActor](https://developer.apple.com/documentation/Swift/MainActor)
func setBackgroundImage(
    _ image: [UIImage](https://developer.apple.com/documentation/uikit/uiimage)?,
    for state: [UIControl](https://developer.apple.com/documentation/uikit/uicontrol).[State](https://developer.apple.com/documentation/uikit/uicontrol/state-swift.struct)
)
```

## [ Parameters ](https://developer.apple.com/documentation/uikit/uibutton/setbackgroundimage\(_:for:\)#parameters) 

`image`
    
The background image to use for the specified state. 

`state`
    
The state that uses the specified image. The values are described in [`UIControl.State`](https://developer.apple.com/documentation/uikit/uicontrol/state-swift.struct).
## [Discussion](https://developer.apple.com/documentation/uikit/uibutton/setbackgroundimage\(_:for:\)#Discussion)
In general, if a property is not specified for a state, the default is to use the [`normal`](https://developer.apple.com/documentation/uikit/uicontrol/state-swift.struct/normal) value. If the [`normal`](https://developer.apple.com/documentation/uikit/uicontrol/state-swift.struct/normal) value is not set, then the property defaults to a system value. Therefore, at a minimum, you should set the value for the normal state.
