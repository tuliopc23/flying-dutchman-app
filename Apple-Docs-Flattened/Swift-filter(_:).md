# filter(_:)
Returns a new collection of the same type containing, in order, the elements of the original collection that satisfy the given predicate.
iOS 8.0+iPadOS 8.0+Mac Catalyst 13.0+macOS 10.10+tvOS 9.0+visionOS 1.0+watchOS 2.0+Swift 4.0+
```
func filter(_ isIncluded: (Self.[Element](https://developer.apple.com/documentation/swift/sequence/element)) throws -> [Bool](https://developer.apple.com/documentation/swift/bool)) rethrows -> Self
```

## [ Parameters ](https://developer.apple.com/documentation/Swift/String/filter\(_:\)#parameters) 

`isIncluded`
    
A closure that takes an element of the sequence as its argument and returns a Boolean value indicating whether the element should be included in the returned collection.
## [Return Value](https://developer.apple.com/documentation/Swift/String/filter\(_:\)#return-value)
A collection of the elements that `isIncluded` allowed.
## [Discussion](https://developer.apple.com/documentation/Swift/String/filter\(_:\)#discussion)
In this example, `filter(_:)` is used to include only names shorter than five characters.
```
let cast = ["Vivien", "Marlon", "Kim", "Karl"]
let shortNames = cast.filter { $0.count < 5 }
print(shortNames)
// Prints "["Kim", "Karl"]"

```

Complexity
O(_n_), where _n_ is the length of the collection.
