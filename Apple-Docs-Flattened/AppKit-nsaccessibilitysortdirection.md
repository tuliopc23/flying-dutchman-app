# NSAccessibilitySortDirection
Values that indicate the sort direction of a column.
macOS 10.10+
```
enum NSAccessibilitySortDirection
```

## [Overview](https://developer.apple.com/documentation/appkit/nsaccessibilitysortdirection#overview)
This constant is used with the [`accessibilitySortDirection`](https://developer.apple.com/documentation/appkit/nsaccessibility-c.protocol/accessibilitysortdirection) property.

The column is sorted in ascending values.
[`case descending`](https://developer.apple.com/documentation/appkit/nsaccessibilitysortdirection/descending)
