# WidgetCenter.UserInfoKey
An object that defines keys for accessing information in a user info dictionary.
iOS 14.0+iPadOS 14.0+Mac CatalystmacOS 11.0+visionOS 26.0+watchOS 9.0+
```
struct UserInfoKey
```

## [Overview](https://developer.apple.com/documentation/widgetkit/widgetcenter/userinfokey#overview)
Note
In Objective-C, use [`WGWidgetUserInfoKeyFamily`](https://developer.apple.com/documentation/widgetkit/wgwidgetuserinfokeyfamily) and [`WGWidgetUserInfoKeyKind`](https://developer.apple.com/documentation/widgetkit/wgwidgetuserinfokeykind) instead.
## [Topics](https://developer.apple.com/documentation/widgetkit/widgetcenter/userinfokey#topics)
### [Describing a widget](https://developer.apple.com/documentation/widgetkit/widgetcenter/userinfokey#Describing-a-widget)
[`static let family: String`](https://developer.apple.com/documentation/widgetkit/widgetcenter/userinfokey/family)
A key you use to access the widget’s family.
[`static let kind: String`](https://developer.apple.com/documentation/widgetkit/widgetcenter/userinfokey/kind)
A key you use to access the widget’s kind. The value matches the `kind` property specified in the widget’s configuration.
### [Describing a Live Activity](https://developer.apple.com/documentation/widgetkit/widgetcenter/userinfokey#Describing-a-Live-Activity)
[`static let activityID: String`](https://developer.apple.com/documentation/widgetkit/widgetcenter/userinfokey/activityid)
A key you use to access the activity ID if the widget represents a Live Activity.
