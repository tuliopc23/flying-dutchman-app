# Fonts
Manage the fonts used to display text.
## [Overview](https://developer.apple.com/documentation/appkit/fonts#overview)
The [`NSFont`](https://developer.apple.com/documentation/appkit/nsfont) and [`NSFontManager`](https://developer.apple.com/documentation/appkit/nsfontmanager) classes encapsulate and manage font families, sizes, and variations. The [`NSFont`](https://developer.apple.com/documentation/appkit/nsfont) class defines a single object for each distinct font; for efficiency, these objects, which can be rather large, are shared by all the objects in your app. The [`NSFontPanel`](https://developer.apple.com/documentation/appkit/nsfontpanel) class defines the font specification panel that’s presented to the user.
## [Topics](https://developer.apple.com/documentation/appkit/fonts#topics)
### [Font Data](https://developer.apple.com/documentation/appkit/fonts#Font-Data)
[`class NSFont`](https://developer.apple.com/documentation/appkit/nsfont)
The representation of a font in an app.
[`class NSFontDescriptor`](https://developer.apple.com/documentation/appkit/nsfontdescriptor)
A dictionary of attributes that describe a font.
[`struct NSFontTraitMask`](https://developer.apple.com/documentation/appkit/nsfonttraitmask)
Constants for isolating specific traits of a font.
[`typealias NSFontFamilyClass`](https://developer.apple.com/documentation/appkit/nsfontfamilyclass)
Constants that classify certain stylistic qualities of the font.
[`struct SymbolicTraits`](https://developer.apple.com/documentation/appkit/nsfontdescriptor/symbolictraits-swift.struct)
A symbolic description of the stylistic aspects of a font.
[`class NSFontAssetRequest`](https://developer.apple.com/documentation/appkit/nsfontassetrequest)
[`typealias NSFontSymbolicTraits`](https://developer.apple.com/documentation/appkit/nsfontsymbolictraits)
A symbolic description of stylistic aspects of a font.
### [Management](https://developer.apple.com/documentation/appkit/fonts#Management)
[`class NSFontManager`](https://developer.apple.com/documentation/appkit/nsfontmanager)
The center of activity for the font-conversion system.
[`class NSFontCollection`](https://developer.apple.com/documentation/appkit/nsfontcollection)
A font collection, which is a group of font descriptors taken together as a single object.
[`class NSMutableFontCollection`](https://developer.apple.com/documentation/appkit/nsmutablefontcollection)
A mutable collection of font descriptors taken together as a single object.
[`struct NSFontCollectionOptions`](https://developer.apple.com/documentation/appkit/nsfontcollectionoptions)
Constants that support font collection management.
