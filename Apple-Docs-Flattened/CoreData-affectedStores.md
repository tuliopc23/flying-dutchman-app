Instance Property
# affectedStores
An array of persistent stores specified for the fetch request.
iOS 3.0+iPadOS 3.0+Mac Catalyst 13.1+macOS 10.4+tvOSvisionOS 1.0+watchOS 2.0+
```
var affectedStores: [[NSPersistentStore](https://developer.apple.com/documentation/coredata/nspersistentstore)]? { get set }
```

## [Discussion](https://developer.apple.com/documentation/CoreData/NSFetchRequest/affectedStores#Discussion)
The contents of the array are the identifiers for the stores to be searched when the fetch request is executed.
