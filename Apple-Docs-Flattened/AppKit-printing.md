# Printing
Display the system print panels and manage the printing process.
## [Overview](https://developer.apple.com/documentation/appkit/printing#overview)
For design guidance, see [Human Interface Guidelines](https://developer.apple.com/design/human-interface-guidelines/macos/system-capabilities/printing/).
## [Topics](https://developer.apple.com/documentation/appkit/printing#topics)
### [Print Panels](https://developer.apple.com/documentation/appkit/printing#Print-Panels)
[`class NSPrintPanel`](https://developer.apple.com/documentation/appkit/nsprintpanel)
The Print panel that queries the user for information about a print job.
[`class NSPageLayout`](https://developer.apple.com/documentation/appkit/nspagelayout)
A panel that queries the user for information such as paper type and orientation.
### [Print Information](https://developer.apple.com/documentation/appkit/printing#Print-Information)
[`class NSPrinter`](https://developer.apple.com/documentation/appkit/nsprinter)
An object that describes a printer’s capabilities.
[`class NSPrintInfo`](https://developer.apple.com/documentation/appkit/nsprintinfo)
An object that stores information that’s used to generate printed output.
[`class NSPrintOperation`](https://developer.apple.com/documentation/appkit/nsprintoperation)
An object that controls operations that generate Encapsulated PostScript (EPS) code, Portable Document Format (PDF) code, or print jobs.
## [See Also](https://developer.apple.com/documentation/appkit/printing#see-also)
### [Graphics, Drawing, Color, and Printing](https://developer.apple.com/documentation/appkit/printing#Graphics-Drawing-Color-and-Printing)
[API Reference Images and PDF](https://developer.apple.com/documentation/appkit/images-and-pdf)
Create and manage images, in bitmap, PDF, and other formats.
[API Reference Drawing](https://developer.apple.com/documentation/appkit/drawing)
Draw shapes, images, and other content on the screen.
[API Reference Color](https://developer.apple.com/documentation/appkit/color)
Represent colors using built-in or custom formats, and give users options for selecting and applying colors.
