# Subject
A publisher that exposes a method for outside callers to publish elements.
iOS 13.0+iPadOS 13.0+Mac Catalyst 13.0+macOS 10.15+tvOS 13.0+visionOS 1.0+watchOS 6.0+
```
protocol Subject<[Output](https://developer.apple.com/documentation/combine/publisher/output), [Failure](https://developer.apple.com/documentation/combine/publisher/failure)> : AnyObject, [Publisher](https://developer.apple.com/documentation/combine/publisher)
```

## [Overview](https://developer.apple.com/documentation/Combine/Subject#overview)
A subject is a publisher that you can use to ”inject” values into a stream, by calling its [`send(_:)`](https://developer.apple.com/documentation/combine/subject/send\(_:\)) method. This can be useful for adapting existing imperative code to the Combine model.
## [Topics](https://developer.apple.com/documentation/Combine/Subject#topics)

**Required**
[`func send(completion: Subscribers.Completion<Self.Failure>)`](https://developer.apple.com/documentation/combine/subject/send\(completion:\))
Sends a completion signal to the subscriber.
