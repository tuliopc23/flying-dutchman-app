# init(expanded:compactLeading:compactTrailing:minimal:)
Creates a configuration object with views that appear in the Dynamic Island.
iOS 16.1+iPadOS 16.1+
```
init<Expanded, CompactLeading, CompactTrailing, Minimal>(
    @[DynamicIslandExpandedContentBuilder](https://developer.apple.com/documentation/widgetkit/dynamicislandexpandedcontentbuilder) expanded: @escaping () -> [DynamicIslandExpandedContent](https://developer.apple.com/documentation/widgetkit/dynamicislandexpandedcontent)<Expanded>,
    @[ViewBuilder](https://developer.apple.com/documentation/SwiftUI/ViewBuilder) compactLeading: @escaping () -> CompactLeading,
    @[ViewBuilder](https://developer.apple.com/documentation/SwiftUI/ViewBuilder) compactTrailing: @escaping () -> CompactTrailing,
    @[ViewBuilder](https://developer.apple.com/documentation/SwiftUI/ViewBuilder) minimal: @escaping () -> Minimal
) where Expanded : [View](https://developer.apple.com/documentation/SwiftUI/View), CompactLeading : [View](https://developer.apple.com/documentation/SwiftUI/View), CompactTrailing : [View](https://developer.apple.com/documentation/SwiftUI/View), Minimal : [View](https://developer.apple.com/documentation/SwiftUI/View)
```

## [ Parameters ](https://developer.apple.com/documentation/widgetkit/dynamicisland/init\(expanded:compactleading:compacttrailing:minimal:\)#parameters) 

`expanded`
    
A closure that builds the view for the expanded presentation of the Live Activity. 

`compactLeading`
    
A closure that builds the view for the compact leading presentation of the Live Activity. 

`compactTrailing`
    
A closure that builds the view for the compact trailing presentation of the Live Activity. 

`minimal`
    
A closure that builds the view for the minimal presentation of the Live Activity.
