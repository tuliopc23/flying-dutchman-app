# init(code:locale:)
Creates a decimal currency format style that uses the given currency code and locale.
iOS 15.0+iPadOS 15.0+Mac Catalyst 15.0+macOS 12.0+tvOS 15.0+visionOS 1.0+watchOS 8.0+
```
init(
    code: [String](https://developer.apple.com/documentation/Swift/String),
    locale: [Locale](https://developer.apple.com/documentation/foundation/locale) = .autoupdatingCurrent
)
```

## [ Parameters ](https://developer.apple.com/documentation/Foundation/Decimal/FormatStyle/Currency/init\(code:locale:\)#parameters) 

`code`
    
The currency code to use, such as `EUR` or `JPY`. 

`locale`
    
The locale to use when formatting or parsing decimal values. Defaults to [`autoupdatingCurrent`](https://developer.apple.com/documentation/foundation/locale/autoupdatingcurrent).
