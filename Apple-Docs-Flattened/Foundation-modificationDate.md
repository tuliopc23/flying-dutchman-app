# modificationDate
The modification date of the version.
iOS 5.0+iPadOS 5.0+Mac Catalyst 13.1+macOS 10.7+tvOS 9.0+visionOS 1.0+watchOS 2.0+
```
var modificationDate: [Date](https://developer.apple.com/documentation/foundation/date)? { get }
```

## [Discussion](https://developer.apple.com/documentation/Foundation/NSFileVersion/modificationDate#Discussion)
If the version has been deleted, this value is `nil`.
