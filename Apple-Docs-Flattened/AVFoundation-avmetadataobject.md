# AVMetadataObject
The abstract superclass for objects provided by a metadata capture output.
iOS 6.0+iPadOS 6.0+Mac Catalyst 14.0+macOS 10.10+tvOS 9.0+
```
class AVMetadataObject
```

## [Overview](https://developer.apple.com/documentation/avfoundation/avmetadataobject#overview)
The `AVMetadataObject` class is an abstract class that defines the basic properties associated with a piece of metadata. These attributes reflect information either about the metadata itself or the media from which the metadata originated. Subclasses are responsible for providing appropriate values for each of the relevant properties.
You shouldn’t subclass `AVMetadataObject` directly. Instead, you use one of the defined subclasses provided by the AVFoundation framework. Similarly, you don’t create instances of this class yourself but use an [`AVCaptureMetadataOutput`](https://developer.apple.com/documentation/avfoundation/avcapturemetadataoutput) object to retrieve them from the captured data.

The bounding rectangle associated with the metadata.
[`var duration: CMTime`](https://developer.apple.com/documentation/avfoundation/avmetadataobject/duration)
The duration of the media associated with this metadata object.
[`var time: CMTime`](https://developer.apple.com/documentation/avfoundation/avmetadataobject/time)
The media time value associated with the metadata object.
[`var type: AVMetadataObject.ObjectType`](https://developer.apple.com/documentation/avfoundation/avmetadataobject/type)
The type of metadata that this object provides.
[`struct ObjectType`](https://developer.apple.com/documentation/avfoundation/avmetadataobject/objecttype)
Constants that identify metadata object types.
[`var isFixedFocus: Bool`](https://developer.apple.com/documentation/avfoundation/avmetadataobject/isfixedfocus)
A BOOL indicating whether this metadata object represents a fixed focus.
[`var cinematicVideoFocusMode: AVCaptureDevice.CinematicVideoFocusMode`](https://developer.apple.com/documentation/avfoundation/avmetadataobject/cinematicvideofocusmode)
The current focus mode when an object is detected during a Cinematic Video recording.
[`var groupID: Int`](https://developer.apple.com/documentation/avfoundation/avmetadataobject/groupid)
An identifier associated with a metadata object used to group it with other metadata objects belonging to a common parent.
[`var objectID: Int`](https://developer.apple.com/documentation/avfoundation/avmetadataobject/objectid)
A unique identifier for each detected object type (face, body, hands, heads and salient objects) in a collection.
