# Enriching your text in text views
Add exclusion paths, text attachments, and text lists to your text, and render it with text views.
[ Download ](https://docs-assets.developer.apple.com/published/78ef49b6f6d9/EnrichingYourTextInTextViews.zip)
iOS 16.0+iPadOS 16.0+macOS 13.0+Xcode 14.0+

This sample code project is associated with WWDC22 session [10090: What’s new in TextKit and text views](https://developer.apple.com/wwdc22/10090/).

A class that represents a text list node.

An abstract base class that represents the smallest units of text layout such as paragraphs or attachments.
