# kSecAttrAccessibleWhenPasscodeSetThisDeviceOnly
The data in the keychain can only be accessed when the device is unlocked. Only available if a passcode is set on the device.
iOS 8.0+iPadOS 8.0+Mac Catalyst 13.1+macOS 10.10+tvOS 9.0+visionOS 1.0+watchOS 2.0+
```
let kSecAttrAccessibleWhenPasscodeSetThisDeviceOnly: [CFString](https://developer.apple.com/documentation/CoreFoundation/CFString)
```

## [Discussion](https://developer.apple.com/documentation/Security/kSecAttrAccessibleWhenPasscodeSetThisDeviceOnly#Discussion)
This is recommended for items that only need to be accessible while the application is in the foreground. Items with this attribute never migrate to a new device. After a backup is restored to a new device, these items are missing. No items can be stored in this class on devices without a passcode. Disabling the device passcode causes all items in this class to be deleted.
