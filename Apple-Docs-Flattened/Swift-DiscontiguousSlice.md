# DiscontiguousSlice
A collection wrapper that provides access to the elements of a collection, indexed by a set of indices.
iOS 18.0+iPadOS 18.0+Mac Catalyst 18.0+macOS 15.0+tvOS 18.0+visionOS 2.0+watchOS 11.0+
```
struct DiscontiguousSlice<Base> where Base : [Collection](https://developer.apple.com/documentation/swift/collection)
```
