# doubleValue
The value of the receiver’s cell as a double-precision floating-point number.
macOS
```
@[MainActor](https://developer.apple.com/documentation/Swift/MainActor)
var doubleValue: [Double](https://developer.apple.com/documentation/Swift/Double) { get set }
```

## [Discussion](https://developer.apple.com/documentation/appkit/nscontrol/doublevalue#Discussion)
If the control contains many cells (for example, `NSMatrix`), then this property contains the value of the currently selected cell. If the control is in the process of editing the affected cell, then it invokes the [`validateEditing()`](https://developer.apple.com/documentation/appkit/nscontrol/validateediting\(\)) method before getting the value.
If the cell is being edited, setting this property aborts all editing before setting the value. If the cell does not inherit from `NSActionCell`, setting this property marks the cell’s interior as needing to be redisplayed; `NSActionCell` performs its own updating of cells.
