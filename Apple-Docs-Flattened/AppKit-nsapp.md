# NSApp
The global variable for the shared app instance.
macOS
```
@[MainActor](https://developer.apple.com/documentation/Swift/MainActor)
var NSApp: [NSApplication](https://developer.apple.com/documentation/appkit/nsapplication)!
```

## [Discussion](https://developer.apple.com/documentation/appkit/nsapp#Discussion)
The value in this global variable is the same as accessing the [`shared`](https://developer.apple.com/documentation/appkit/nsapplication/shared) property of the app.
## [See Also](https://developer.apple.com/documentation/appkit/nsapp#see-also)
