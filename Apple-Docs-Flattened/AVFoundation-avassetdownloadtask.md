# AVAssetDownloadTask
A session used to download HTTP Live Streaming assets.
iOS 9.0+iPadOS 9.0+Mac Catalyst 13.1+macOS 10.15+visionOS 1.0+watchOS 10.0+
```
class AVAssetDownloadTask
```

## [Overview](https://developer.apple.com/documentation/avfoundation/avassetdownloadtask#overview)
This class is a subclass of [`URLSessionTask`](https://developer.apple.com/documentation/Foundation/URLSessionTask) that you use to download HTTP Live Streaming assets. You create instances of this class by calling [`makeAssetDownloadTask(downloadConfiguration:)`](https://developer.apple.com/documentation/avfoundation/avassetdownloadurlsession/makeassetdownloadtask\(downloadconfiguration:\)) on the download session.
