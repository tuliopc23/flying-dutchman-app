# IntentParameter
A property wrapper that indicates the associated property is an input argument of the app intent.
iOS 16.0+iPadOS 16.0+Mac CatalystmacOS 13.0+tvOS 16.0+visionOSwatchOS 9.0+
```
@propertyWrapper
final class IntentParameter<Value> where Value : _IntentValue, Value : [Sendable](https://developer.apple.com/documentation/Swift/Sendable)
```
