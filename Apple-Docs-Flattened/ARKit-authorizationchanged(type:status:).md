# ARKitSession.Event.authorizationChanged(type:status:)
An event that represents a change in authorization status for a specific authorization type.
visionOS 1.0+
```
case authorizationChanged(
    type: [ARKitSession](https://developer.apple.com/documentation/arkit/arkitsession).[AuthorizationType](https://developer.apple.com/documentation/arkit/arkitsession/authorizationtype),
    status: [ARKitSession](https://developer.apple.com/documentation/arkit/arkitsession).[AuthorizationStatus](https://developer.apple.com/documentation/arkit/arkitsession/authorizationstatus)
)
```

## [ Parameters ](https://developer.apple.com/documentation/arkit/arkitsession/event/authorizationchanged\(type:status:\)#parameters) 

`type`
    
The type of authorization status that changed. 

`status`
    
The new state of authorization.
