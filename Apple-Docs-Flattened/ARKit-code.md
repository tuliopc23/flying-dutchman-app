### [Errors](https://developer.apple.com/documentation/arkit/arerror/code#Errors)
[`case requestFailed`](https://developer.apple.com/documentation/arkit/arerror/code/requestfailed)
An error that indicates a request fails.
[`case cameraUnauthorized`](https://developer.apple.com/documentation/arkit/arerror/code/cameraunauthorized)
An error that indicates the app lacks user permission for the camera.
[`case fileIOFailed`](https://developer.apple.com/documentation/arkit/arerror/code/fileiofailed)
An error that indicates a file access fails to read or write.
[`case insufficientFeatures`](https://developer.apple.com/documentation/arkit/arerror/code/insufficientfeatures)
An error that indicates the framework requires more features to complete a task.
[`case invalidCollaborationData`](https://developer.apple.com/documentation/arkit/arerror/code/invalidcollaborationdata)
An error that indicates the framework fails to parse collaboration data the app receives over the network.
[`case invalidConfiguration`](https://developer.apple.com/documentation/arkit/arerror/code/invalidconfiguration)
An error that indicates the configuration contains ambiguous or erroneous data.
[`case invalidReferenceImage`](https://developer.apple.com/documentation/arkit/arerror/code/invalidreferenceimage)
An error that indicates the framework fails to process a reference image.
[`case invalidReferenceObject`](https://developer.apple.com/documentation/arkit/arerror/code/invalidreferenceobject)
An error that indicates the framework fails to process a reference object.
[`case invalidWorldMap`](https://developer.apple.com/documentation/arkit/arerror/code/invalidworldmap)
An error that indicates the framework fails to process a world map.
[`case microphoneUnauthorized`](https://developer.apple.com/documentation/arkit/arerror/code/microphoneunauthorized)
An error that indicates the app lacks user permission for the microphone.
[`case objectMergeFailed`](https://developer.apple.com/documentation/arkit/arerror/code/objectmergefailed)
An error that indicates the framework fails to merge a detected object.
[`case sensorFailed`](https://developer.apple.com/documentation/arkit/arerror/code/sensorfailed)
An error that indicates a sensor fails to provide required input.
[`case sensorUnavailable`](https://developer.apple.com/documentation/arkit/arerror/code/sensorunavailable)
An error that indicates the framework fails to access a required sensor.
[`case unsupportedConfiguration`](https://developer.apple.com/documentation/arkit/arerror/code/unsupportedconfiguration)
An error that indicates the device lacks support for the session’s configuration.
[`case worldTrackingFailed`](https://developer.apple.com/documentation/arkit/arerror/code/worldtrackingfailed)
An error that indicates when world tracking experiences an unrecoverable problem.
[`case geoTrackingFailed`](https://developer.apple.com/documentation/arkit/arerror/code/geotrackingfailed)
An error that indicates when localization imagery fails to match the device’s camera captures.
[`case geoTrackingNotAvailableAtLocation`](https://developer.apple.com/documentation/arkit/arerror/code/geotrackingnotavailableatlocation)
An error that indicates a location lacks geotracking support.
[`case locationUnauthorized`](https://developer.apple.com/documentation/arkit/arerror/code/locationunauthorized)
An error that indicates the app lacks user permission for the device’s current location.
[`case highResolutionFrameCaptureFailed`](https://developer.apple.com/documentation/arkit/arerror/code/highresolutionframecapturefailed)
An error that indicates a problem in the system’s capture pipeline.
[`case highResolutionFrameCaptureInProgress`](https://developer.apple.com/documentation/arkit/arerror/code/highresolutionframecaptureinprogress)
An error that indicates the system needs to finish a high-resolution frame request before accepting another request.
