# Persistent history
Use persistent history tracking to determine what changes have occurred in the store since the enabling of persistent history tracking.
## [Topics](https://developer.apple.com/documentation/coredata/persistent-history#topics)
### [Tracking History](https://developer.apple.com/documentation/coredata/persistent-history#Tracking-History)
[`class NSPersistentHistoryToken`](https://developer.apple.com/documentation/coredata/nspersistenthistorytoken)
A bookmark for keeping track the most recent history that you’ve processed.
### [Requesting History](https://developer.apple.com/documentation/coredata/persistent-history#Requesting-History)
[`class NSPersistentHistoryChangeRequest`](https://developer.apple.com/documentation/coredata/nspersistenthistorychangerequest)
A request to fetch or purge persistent history.
[`class NSPersistentHistoryResult`](https://developer.apple.com/documentation/coredata/nspersistenthistoryresult)
The result of a request to fetch persistent history.
### [Reading History](https://developer.apple.com/documentation/coredata/persistent-history#Reading-History)
[`class NSPersistentHistoryTransaction`](https://developer.apple.com/documentation/coredata/nspersistenthistorytransaction)
A set of changes in the persistent history based on a context save or batch operation.
[`class NSPersistentHistoryChange`](https://developer.apple.com/documentation/coredata/nspersistenthistorychange)
A change representing the insertion, update, or deletion of a managed object in the persistent store.
## [See Also](https://developer.apple.com/documentation/coredata/persistent-history#see-also)
### [Change processing](https://developer.apple.com/documentation/coredata/persistent-history#Change-processing)
[Accessing data when the store changes](https://developer.apple.com/documentation/coredata/accessing-data-when-the-store-changes)
Guarantee that a context won’t see store changes until you tell it to look.
[Consuming relevant store changes](https://developer.apple.com/documentation/coredata/consuming-relevant-store-changes)
Filter store transactions for changes relevant to the current view.
