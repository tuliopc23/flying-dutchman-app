# Bonjour
Advertise services for easy discovery on local networks, or discover services advertised by others.
## [Topics](https://developer.apple.com/documentation/Foundation/bonjour#topics)
### [Local Network Services](https://developer.apple.com/documentation/Foundation/bonjour#Local-Network-Services)
[`class NetService`](https://developer.apple.com/documentation/foundation/netservice)
A network service that broadcasts its availability using multicast DNS.
Deprecated
[`protocol NetServiceDelegate`](https://developer.apple.com/documentation/foundation/netservicedelegate)
The interface a net service uses to inform its delegate about the state of the service it offers.
[`NSBonjourServices`](https://developer.apple.com/documentation/BundleResources/Information-Property-List/NSBonjourServices)
Bonjour service types browsed by the app.
[`NSLocalNetworkUsageDescription`](https://developer.apple.com/documentation/BundleResources/Information-Property-List/NSLocalNetworkUsageDescription)
A message that tells people why the app is requesting access to the local network.
### [Service Discovery](https://developer.apple.com/documentation/Foundation/bonjour#Service-Discovery)
[`class NetServiceBrowser`](https://developer.apple.com/documentation/foundation/netservicebrowser)
A network service browser that finds published services on a network using multicast DNS.
Deprecated
[`protocol NetServiceBrowserDelegate`](https://developer.apple.com/documentation/foundation/netservicebrowserdelegate)
The interface a net service browser uses to inform a delegate about the state of service discovery.
