# tableView(_:moveRowAt:to:)
Tells the data source to move a row at a specific location in the table view to another location.
iOS 8.0+iPadOS 8.0+Mac Catalyst 13.1+tvOS 9.0+visionOS 1.0+
```
@[MainActor](https://developer.apple.com/documentation/Swift/MainActor)
optional func tableView(
    _ tableView: [UITableView](https://developer.apple.com/documentation/uikit/uitableview),
    moveRowAt sourceIndexPath: [IndexPath](https://developer.apple.com/documentation/Foundation/IndexPath),
    to destinationIndexPath: [IndexPath](https://developer.apple.com/documentation/Foundation/IndexPath)
)
```

## [ Parameters ](https://developer.apple.com/documentation/uikit/uitableviewdatasource/tableview\(_:moverowat:to:\)#parameters) 

`tableView`
    
The table-view object requesting this action. 

`sourceIndexPath`
    
An index path locating the row to be moved in `tableView`. 

`destinationIndexPath`
    
An index path locating the row in `tableView` that’s the destination of the move.
## [Discussion](https://developer.apple.com/documentation/uikit/uitableviewdatasource/tableview\(_:moverowat:to:\)#Discussion)
The [`UITableView`](https://developer.apple.com/documentation/uikit/uitableview) object sends this message to the data source when the user presses the reorder control in the row at `sourceIndexPath`.
