Instance Method
# splitViewItem(for:)
Returns the corresponding split view item for the specified child view controller of the split view controller.
macOS 10.10+
```
@[MainActor](https://developer.apple.com/documentation/Swift/MainActor)
func splitViewItem(for viewController: [NSViewController](https://developer.apple.com/documentation/appkit/nsviewcontroller)) -> [NSSplitViewItem](https://developer.apple.com/documentation/appkit/nssplitviewitem)?
```

## [ Parameters ](https://developer.apple.com/documentation/appkit/nssplitviewcontroller/splitviewitem\(for:\)#parameters) 

`viewController`
    
The child view controller with the corresponding split view item you want.
## [Return Value](https://developer.apple.com/documentation/appkit/nssplitviewcontroller/splitviewitem\(for:\)#return-value)
The corresponding split view item, or `nil` if `viewController` isn’t a child of the split view controller.
