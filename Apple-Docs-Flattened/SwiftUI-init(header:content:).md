# init(header:content:)
Creates a section with a header and the provided section content.
iOS 13.0–26.2DeprecatediPadOS 13.0–26.2DeprecatedMac Catalyst 13.0–26.2DeprecatedmacOS 10.15–26.2DeprecatedtvOS 13.0–26.2DeprecatedvisionOS 1.0–26.2DeprecatedwatchOS 6.0–26.2Deprecated
```
init(
    header: Parent,
    @[ViewBuilder](https://developer.apple.com/documentation/swiftui/viewbuilder) content: () -> Content
)
```

Available when `Parent` conforms to `View`, `Content` conforms to `View`, and `Footer` is `EmptyView`.
Deprecated
Use [`init(content:header:)`](https://developer.apple.com/documentation/swiftui/section/init\(content:header:\)-29e2l) instead.
## [ Parameters ](https://developer.apple.com/documentation/swiftui/section/init\(header:content:\)#parameters) 

`header`
    
A view to use as the section’s header. 

`content`
    
The section’s content.
