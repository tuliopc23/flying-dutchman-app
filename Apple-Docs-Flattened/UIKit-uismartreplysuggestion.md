# UISmartReplySuggestion
A class you use to handle a Smart Reply suggestion.
iOS 18.4+iPadOS 18.4+
```
class UISmartReplySuggestion
```

## [Overview](https://developer.apple.com/documentation/uikit/uismartreplysuggestion#overview)
Use the [`smartReply`](https://developer.apple.com/documentation/uikit/uismartreplysuggestion/smartreply) string as a signal of the user’s intention when you generate long form text based on the option the user selected.
## [Topics](https://developer.apple.com/documentation/uikit/uismartreplysuggestion#topics)

A string from the Smart Reply option the user selected.
