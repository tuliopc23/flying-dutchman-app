# uniformTopRadius(_:bottomLeftRadius:bottomRightRadius:)
A configuration that applies the top radius you provide to the top corners, with optional independent radii for the bottom corners.
iOS 26.0+iPadOS 26.0+Mac CatalysttvOS 26.0+visionOS 26.0+
```
static func uniformTopRadius(
    _ topRadius: [UICornerRadius](https://developer.apple.com/documentation/uikit/uicornerradius-swift.struct),
    bottomLeftRadius: [UICornerRadius](https://developer.apple.com/documentation/uikit/uicornerradius-swift.struct)? = nil,
    bottomRightRadius: [UICornerRadius](https://developer.apple.com/documentation/uikit/uicornerradius-swift.struct)? = nil
) -> [UICornerConfiguration](https://developer.apple.com/documentation/uikit/uicornerconfiguration-swift.struct)
```

## [ Parameters ](https://developer.apple.com/documentation/uikit/uicornerconfiguration-swift.struct/uniformtopradius\(_:bottomleftradius:bottomrightradius:\)#parameters) 

`topRadius`
    
A [`UICornerRadius`](https://developer.apple.com/documentation/uikit/uicornerradius-swift.struct) that represents the radius to use for the top-left and top-right corners. 

`bottomLeftRadius`
    
An optional [`UICornerRadius`](https://developer.apple.com/documentation/uikit/uicornerradius-swift.struct) that represents the radius to use for the bottom-left corner. 

`bottomRightRadius`
    
An optional [`UICornerRadius`](https://developer.apple.com/documentation/uikit/uicornerradius-swift.struct) that represents the radius to use for the bottom-right corner.
