## [ Parameters ](https://developer.apple.com/documentation/uikit/uiapplicationdelegate/application\(_:didfailtocontinueuseractivitywithtype:error:\)#parameters) 

`application`
    
Your shared app object. 

`userActivityType`

`error`
    
An error object indicating the reason for the failure.
## [Discussion](https://developer.apple.com/documentation/uikit/uiapplicationdelegate/application\(_:didfailtocontinueuseractivitywithtype:error:\)#Discussion)
Use this method to let the user know that the specified activity could not be continued. If you do not implement this method, UIKit displays an error to the user with an appropriate message about the reason for the failure.
This method is not called if either [`application(_:willFinishLaunchingWithOptions:)`](https://developer.apple.com/documentation/uikit/uiapplicationdelegate/application\(_:willfinishlaunchingwithoptions:\)) or [`application(_:didFinishLaunchingWithOptions:)`](https://developer.apple.com/documentation/uikit/uiapplicationdelegate/application\(_:didfinishlaunchingwithoptions:\)) returns [`false`](https://developer.apple.com/documentation/Swift/false).
