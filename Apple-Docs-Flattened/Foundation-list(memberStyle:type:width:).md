# list(memberStyle:type:width:)
Returns a format style to format a list of items.
iOS 15.0+iPadOS 15.0+Mac Catalyst 15.0+macOS 12.0+tvOS 15.0+visionOS 1.0+watchOS 8.0+
```
static func list<MemberStyle, Base>(
    memberStyle: MemberStyle,
    type: [ListFormatStyle](https://developer.apple.com/documentation/foundation/listformatstyle)<MemberStyle, Base>.[ListType](https://developer.apple.com/documentation/foundation/listformatstyle/listtype-swift.enum),
    width: [ListFormatStyle](https://developer.apple.com/documentation/foundation/listformatstyle)<MemberStyle, Base>.[Width](https://developer.apple.com/documentation/foundation/listformatstyle/width-swift.enum) = .standard
) -> Self where Self == [ListFormatStyle](https://developer.apple.com/documentation/foundation/listformatstyle)<MemberStyle, Base>, MemberStyle : [FormatStyle](https://developer.apple.com/documentation/foundation/formatstyle), Base : [Sequence](https://developer.apple.com/documentation/Swift/Sequence), MemberStyle.[FormatInput](https://developer.apple.com/documentation/foundation/formatstyle/formatinput) == Base.[Element](https://developer.apple.com/documentation/Swift/Sequence/Element), MemberStyle.[FormatOutput](https://developer.apple.com/documentation/foundation/formatstyle/formatoutput) == [String](https://developer.apple.com/documentation/Swift/String)
```

## [ Parameters ](https://developer.apple.com/documentation/Foundation/FormatStyle/list\(memberStyle:type:width:\)#parameters) 

`memberStyle`
    
The format style to apply to each item in the list. 

`type`
    
The list type to apply, such as cumulative ([`ListFormatStyle.ListType.and`](https://developer.apple.com/documentation/foundation/listformatstyle/listtype-swift.enum/and)) or alternative ([`ListFormatStyle.ListType.or`](https://developer.apple.com/documentation/foundation/listformatstyle/listtype-swift.enum/or)) elements. 

`width`
    
The width to use when formatting, such as [`ListFormatStyle.Width.standard`](https://developer.apple.com/documentation/foundation/listformatstyle/width-swift.enum/standard) or [`ListFormatStyle.Width.narrow`](https://developer.apple.com/documentation/foundation/listformatstyle/width-swift.enum/narrow).
## [Return Value](https://developer.apple.com/documentation/Foundation/FormatStyle/list\(memberStyle:type:width:\)#return-value)
A list format style that formats an array as a textual list of items.
## [Discussion](https://developer.apple.com/documentation/Foundation/FormatStyle/list\(memberStyle:type:width:\)#Discussion)
Use the dot-notation form of this type method when the call point allows the use of [`ListFormatStyle`](https://developer.apple.com/documentation/foundation/listformatstyle). You typically do this when calling the [`formatted(_:)`](https://developer.apple.com/documentation/Swift/Sequence/formatted\(_:\)) method of [`Sequence`](https://developer.apple.com/documentation/Swift/Sequence).
The following example creates an array of integers, then uses [`formatted(_:)`](https://developer.apple.com/documentation/Swift/Sequence/formatted\(_:\)) and the list format style provided by this method to format the items. By using a currency [`IntegerFormatStyle`](https://developer.apple.com/documentation/foundation/integerformatstyle), the list format style expresses each member as US dollars. The example also modifies the list format style to use the `en_US` locale, so the resulting string uses US English conventions for commas and conjunctions (“and”).
```
let items: [Int] = [100, 1000, 10000, 100000, 1000000]
let formatted = items.formatted(
    .list(memberStyle: .currency(code: "USD"),
          type: .and)
    .locale(Locale(identifier: "en_US"))) // "$100.00, $1,000.00, $10,000.00, $100,000.00, and $1,000,000.00"

```
