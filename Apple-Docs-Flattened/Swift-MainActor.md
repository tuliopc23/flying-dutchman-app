# MainActor
A singleton actor whose executor is equivalent to the main dispatch queue.
iOS 13.0+iPadOS 13.0+Mac Catalyst 13.0+macOS 10.15+tvOS 13.0+visionOS 1.0+watchOS 6.0+
```
@globalActor
final actor MainActor
```
