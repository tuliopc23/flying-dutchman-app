# FileDocumentConfiguration
The properties of an open file document.
iOS 14.0+iPadOS 14.0+Mac Catalyst 14.0+macOS 11.0+visionOS 1.0+
```
struct FileDocumentConfiguration<Document> where Document : [FileDocument](https://developer.apple.com/documentation/swiftui/filedocument)
```

## [Overview](https://developer.apple.com/documentation/swiftui/filedocumentconfiguration#overview)
You receive an instance of this structure when you create a [`DocumentGroup`](https://developer.apple.com/documentation/swiftui/documentgroup) with a value file type. Use it to access the document in your viewer or editor.
## [Topics](https://developer.apple.com/documentation/swiftui/filedocumentconfiguration#topics)

The current document model.
[`var $document: Binding<Document>`](https://developer.apple.com/documentation/swiftui/filedocumentconfiguration/$document)
### [Getting document properties](https://developer.apple.com/documentation/swiftui/filedocumentconfiguration#Getting-document-properties)
[`var fileURL: URL?`](https://developer.apple.com/documentation/swiftui/filedocumentconfiguration/fileurl)
The URL of the open file document.
[`var isEditable: Bool`](https://developer.apple.com/documentation/swiftui/filedocumentconfiguration/iseditable)
A Boolean that indicates whether you can edit the document.
