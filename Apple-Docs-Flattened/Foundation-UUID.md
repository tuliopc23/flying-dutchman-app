# UUID
A universally unique value to identify types, interfaces, and other items.
iOS 6.0+iPadOS 6.0+Mac Catalyst 6.0+macOS 10.8+tvOS 9.0+visionOS 1.0+watchOS 2.0+
```
struct UUID
```

## [Topics](https://developer.apple.com/documentation/Foundation/UUID#topics)
### [Creating UUIDs](https://developer.apple.com/documentation/Foundation/UUID#Creating-UUIDs)
[`init()`](https://developer.apple.com/documentation/foundation/uuid/init\(\))
Creates a UUID with RFC 4122 version 4 random bytes.
[`init(uuid: uuid_t)`](https://developer.apple.com/documentation/foundation/uuid/init\(uuid:\))
Creates a UUID from the uuid C-language structure.
[`init?(uuidString: String)`](https://developer.apple.com/documentation/foundation/uuid/init\(uuidstring:\))
Creates a UUID from a string representation.
### [Getting UUID Values](https://developer.apple.com/documentation/Foundation/UUID#Getting-UUID-Values)
[`var uuid: (UInt8, UInt8, UInt8, UInt8, UInt8, UInt8, UInt8, UInt8, UInt8, UInt8, UInt8, UInt8, UInt8, UInt8, UInt8, UInt8)`](https://developer.apple.com/documentation/foundation/uuid/uuid)
Returns the UUID as bytes.
[`var uuidString: String`](https://developer.apple.com/documentation/foundation/uuid/uuidstring)
Returns a string created from the UUID, such as “E621E1F8-C36C-495A-93FC-0C247A3E6E5F”
### [Using Reference Types](https://developer.apple.com/documentation/Foundation/UUID#Using-Reference-Types)
[`class NSUUID`](https://developer.apple.com/documentation/foundation/nsuuid)
A universally unique value that can be used to identify types, interfaces, and other items.
