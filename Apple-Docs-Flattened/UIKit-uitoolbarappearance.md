# UIToolbarAppearance
An object for customizing the appearance of a toolbar.
iOS 13.0+iPadOS 13.0+Mac Catalyst 13.1+tvOS 13.0+visionOS 1.0+
```
@[MainActor](https://developer.apple.com/documentation/Swift/MainActor)
class UIToolbarAppearance
```

## [Overview](https://developer.apple.com/documentation/uikit/uitoolbarappearance#overview)
After creating a [`UIToolbarAppearance`](https://developer.apple.com/documentation/uikit/uitoolbarappearance) object, use the methods and properties of this class to specify the appearance of items in the toolbar. Use the inherited properties from [`UIBarAppearance`](https://developer.apple.com/documentation/uikit/uibarappearance) to configure the background and shadow attributes of the toolbar itself.
## [Topics](https://developer.apple.com/documentation/uikit/uitoolbarappearance#topics)
### [Configuring bar button items](https://developer.apple.com/documentation/uikit/uitoolbarappearance#Configuring-bar-button-items)
[`var buttonAppearance: UIBarButtonItemAppearance`](https://developer.apple.com/documentation/uikit/uitoolbarappearance/buttonappearance)
The appearance attributes for plain bar button items in the toolbar.
[`var prominentButtonAppearance: UIBarButtonItemAppearance`](https://developer.apple.com/documentation/uikit/uitoolbarappearance/prominentbuttonappearance)
The appearance attributes for Prominent buttons.
### [Configuring the Done button](https://developer.apple.com/documentation/uikit/uitoolbarappearance#Configuring-the-Done-button)
[`var doneButtonAppearance: UIBarButtonItemAppearance`](https://developer.apple.com/documentation/uikit/uitoolbarappearance/donebuttonappearance)
The appearance attributes for Done buttons.
