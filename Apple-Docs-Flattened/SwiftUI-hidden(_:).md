# hidden(_:)
Hides a toolbar item within its toolbar.
macOS 15.0+
```
nonisolated
func hidden(_ hidden: [Bool](https://developer.apple.com/documentation/Swift/Bool) = true) -> some [ToolbarContent](https://developer.apple.com/documentation/swiftui/toolbarcontent)

```

## [ Parameters ](https://developer.apple.com/documentation/SwiftUI/ToolbarContent/hidden\(_:\)#parameters) 

`hidden`
    
Whether the toolbar item is hidden.
## [Discussion](https://developer.apple.com/documentation/SwiftUI/ToolbarContent/hidden\(_:\)#discussion)
Use this modifier to conditionally display a toolbar item in its toolbar.
```
struct ContentView {
    @State private var showDownloads = false


    var body: some View {
        BrowserView()
            .toolbar {
                ToolbarItem {
                    DownloadsButton()
                }
                .hidden(!showDownloads)
            }
    }
}

```
