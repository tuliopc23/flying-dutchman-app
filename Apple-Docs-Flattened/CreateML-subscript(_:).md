# subscript(_:)
Retrieves or adds an untyped column with the specified name.
iOS 15.0+iPadOS 15.0+Mac Catalyst 15.0+macOS 10.14+tvOS 16.0+visionOS 1.0+
```
subscript(columnName: [String](https://developer.apple.com/documentation/Swift/String)) -> [MLUntypedColumn](https://developer.apple.com/documentation/createml/mluntypedcolumn) { get set }
```

Show all declarations 
## [ Parameters ](https://developer.apple.com/documentation/createml/mldatatable/subscript\(_:\)#parameters) 

`columnName`
    
The name of the column to extract from or add to the datatable.
## [Return Value](https://developer.apple.com/documentation/createml/mldatatable/subscript\(_:\)#return-value)
A new [`MLUntypedColumn`](https://developer.apple.com/documentation/createml/mluntypedcolumn) with the specified name, if it exists; otherwise an invalid column.
## [Overview](https://developer.apple.com/documentation/createml/mldatatable/subscript\(_:\)#overview)
Use this subscript to retrieve an [`MLUntypedColumn`](https://developer.apple.com/documentation/createml/mluntypedcolumn) or add a new one to the data table.
Important
The number of elements in the column must equal the number of rows in the data table. Otherwise, the data table will be invalidated.
![A table on the left with two columns plus a third semi-opaque column.](https://docs-assets.developer.apple.com/published/d47258a4254c249703f7c2ab6c960289/MLDataTable-subscript%28_%3A%29-3wjk-1%402x.png)
For example, to extract, convert, and add a column as shown above, begin by creating a data table.
```
let data: [String: MLDataValueConvertible] = [
    "Planet": ["Mercury", "Venus", "Earth", "Mars", "Jupiter", "Saturn", "Uranus", "Neptune"],
    "Distance (AU)": [0.39, 0.72, 1.00, 1.52, 5.20, 9.54, 19.22, 30.06]
]


var table = try MLDataTable(dictionary: data)

```

After creating the table, extract a column.
```
let distanceInAU = table["Distance (AU)"]

```

Note
Without a type annotation, the compiler uses this version of `subscript(_:)` instead of the equivalent [`subscript(_:)`](https://developer.apple.com/documentation/createml/mldatatable/subscript\(_:\)-5tl9r) from [`MLDataColumn`](https://developer.apple.com/documentation/createml/mldatacolumn).
Use the untyped column’s multiplication operator to create a new column of data.
```
// Create a new column of Doubles by multiplying all the the values in
// `distanceInAU` by 149,597,870.7 (the number of km in an astronomical unit)
let distanceInKm = (distanceInAU * 149_597_870.700)

```

Finally, use the `subscript(_:)` to add the new column to a table with a different name.
```
table["Distance (km)"] = distanceInKm

```

Important
To replace a column, use [`removeColumn(named:)`](https://developer.apple.com/documentation/createml/mldatatable/removecolumn\(named:\)) before adding its replacement. Adding a column with the same name as one already in the data table may invalidate the data table.
