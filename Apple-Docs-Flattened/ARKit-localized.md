# ARGeoTrackingStatus.State.localized
Geo tracking is localized.
iOS 14.0+iPadOS 14.0+Mac Catalyst 14.0+
```
case localized
```

## [Discussion](https://developer.apple.com/documentation/arkit/argeotrackingstatus/state-swift.enum/localized#Discussion)
In [`ARGeoTrackingStatus.State.localized`](https://developer.apple.com/documentation/arkit/argeotrackingstatus/state-swift.enum/localized), ARKit has completed visual localization and the app is free to place location anchors ([`ARGeoAnchor`](https://developer.apple.com/documentation/arkit/argeoanchor)).
