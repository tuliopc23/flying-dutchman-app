# HKWorkoutTypeIdentifier
The workout type identifier.
iOS 8.0+iPadOS 8.0+Mac Catalyst 13.0+macOS 13.0+visionOS 1.0+watchOS 2.0+
```
let HKWorkoutTypeIdentifier: [String](https://developer.apple.com/documentation/Swift/String)
```

## [Discussion](https://developer.apple.com/documentation/healthkit/hkworkouttypeidentifier#Discussion)
Use this type identifier when requesting permission to read or share workout data.
