# object(forInfoDictionaryKey:)
Returns the value associated with the specified key in the receiver’s information property list.
iOS 2.0+iPadOS 2.0+Mac Catalyst 13.0+macOS 10.0+tvOS 9.0+visionOS 1.0+watchOS 2.0+
```
func object(forInfoDictionaryKey key: [String](https://developer.apple.com/documentation/Swift/String)) -> Any?
```

## [ Parameters ](https://developer.apple.com/documentation/Foundation/Bundle/object\(forInfoDictionaryKey:\)#parameters) 

`key`
    
A key in the receiver’s property list.
## [Return Value](https://developer.apple.com/documentation/Foundation/Bundle/object\(forInfoDictionaryKey:\)#return-value)
The value associated with `key` in the receiver’s property list (`Info.plist`). The localized value of a key is returned when one is available.
## [Discussion](https://developer.apple.com/documentation/Foundation/Bundle/object\(forInfoDictionaryKey:\)#Discussion)
Use of this method is preferred over other access methods because it returns the localized value of a key when one is available.
