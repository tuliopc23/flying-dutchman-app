# SliderTick
A representation of a tick in a slider, with associated value and optional label.
iOS 26.0+iPadOS 26.0+Mac Catalyst 26.0+macOS 26.0+visionOS 26.0+watchOS 26.0+
```
struct SliderTick<V> where V : [BinaryFloatingPoint](https://developer.apple.com/documentation/Swift/BinaryFloatingPoint)
```

## [Overview](https://developer.apple.com/documentation/swiftui/slidertick#overview)
The following example shows a slider bound to the value `percentage`. As the slider updates the `currentValueLabel`. The slider also renders marks at a `0.25` step interval.
```
@State private var percentage = 0.5


Slider(value: $percentage) {
    Text("Percentage")
} currentValueLabel: {
    Text("\(percentage)%")
} ticks: {
    SliderTickContentForEach(
        stride(from: 0.0, through: 1.0, by: 0.25).map { $0 },
        id: \.self
    ) { value in
        SliderTick(value) {
            label(for: value)
        }
    }
}

```
